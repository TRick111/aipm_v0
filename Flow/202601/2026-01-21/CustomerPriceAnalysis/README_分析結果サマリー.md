## 4. 主要な分析結論

### 高単価の顧客の特徴

1. **ディナー帯に多い**
   - ランチ帯（11-14時）では低単価が圧倒的に多い
   - ディナー帯（17時以降）では高単価が多い

2. **週末に多い**
   - 土日の高単価比率が平日より約5ポイント高い

3. **追加注文が多い**
   - 総商品数が平均2.5品（低単価は1.3品）
   - アルコール注文率が16.5%（低単価は0.6%）
   - トッピング追加率も高い

### 低単価の顧客の特徴

1. **ランチ帯に集中**
   - 53.4%がランチ帯（11-14時）

2. **平日に多い**
   - 火～木曜日の比率が高い

3. **単品注文が多い**
   - 総商品数が平均1.3品
   - アルコール・トッピングの追加がほとんどない

## 5. 施策への示唆

### 客単価向上のための施策案

1. **ランチ帯の客単価向上**
   - ランチセットへのトッピング追加促進
   - ドリンクセットの訴求強化

2. **ディナー帯の強化**
   - アルコールとのセット提案
   - サイドメニュー・トッピングの追加促進

3. **週末集客の維持・強化**
   - 週末は高単価顧客が多いため、プレミアムメニューの提案

4. **平日ディナーの開拓**
   - 平日でもディナー帯は高単価傾向
   - 仕事帰りの来店促進
