## この会話のスタンス

### ゴールの定義（最重要）

**ゴールは「萌さんを変えること」ではない。**

ゴールは **「自分はこう感じている」を正直に伝えて、萌さんがどう思っているかを聞くこと**。それだけ。

- 伝えた結果、萌さんが「気をつけるね」と言ってくれるかもしれない → ありがたい
- 「私はそういうタイプなんだよね」と言うかもしれない → それも重要な情報。合うか合わないかの判断材料
- 萌さんから「好きだよ」と言ってもらえたら → 最高。でもこれを引き出しにいかない
- どちらの結果でも、**伝えたこと自体が成果**

つまり:
> 「俺はこう思ってるんだけど、萌ちゃんはどう？」
> → 合えば嬉しい。合わないなら、それはそれで受け止める。

**結果に執着しない = 相手の返答で自分の態度を変えない。**
「分かった」と言われてもベタ褒めしない。「うーん」と言われても取り乱さない。どちらでもフラット。

### 最高と最低のシナリオ（事前に腹を括る）

| シナリオ | 内容 | その時の自分 |
|----------|------|-------------|
| 最高 | 自分の気持ちを伝えた上で、萌さんから「好きだよ」と言ってもらえる | 素直に嬉しいと伝える。でも舞い上がりすぎない |
| 中間 | 萌さんが「気をつけるね」or「そういうタイプだから」で終わる | 情報として受け取る。合う合わないはこの先見ていく |
| 最低 | 別れを切り出される | 原因は聞く。でも引き止めない。次に進む |

**最低のシナリオが起きても自分は大丈夫だと、事前に覚悟しておく。**
この覚悟があるからこそ、会話中にフラットでいられる。

### なぜこのスタンスが「余裕」になるか

| スタンス | 相手に伝わる印象 |
|----------|-----------------|
| 「変わってほしい」（結果に執着） | この人は私がいないと不安なんだ → 重い |
| 「俺はこう思ってる。あなたは？」（共有） | この人は自分の考えがあって、でも押しつけない → 対等で魅力的 |
| 「合わないなら仕方ない」（手放せる余裕） | この人は私に依存してない → 安心感がある |

**余裕とは「相手がどう反応しても自分は大丈夫」という態度のこと。**

### 基本方針
- **共有であって要求ではない**: 自分の気持ちを開示するだけ。相手にアクションを強制しない
- **聞くことがメイン**: 伝える:聞く = 3:7 くらいのイメージ。萌さんがどう思っているかを知ることが本当のゴール
- **結果を手放す**: 会話の後にどうなるかは、二人で決まること。自分一人でコントロールしようとしない

### ノンバーバル（振る舞い）

| 場面 | OK | NG |
|------|----|----|
| 会った瞬間 | 笑顔で普通に「久しぶり」。ベタベタしない | 「やっと会えた…」と安堵感を全面に出す |
| 食事中 | リラックスした姿勢。背もたれに寄りかかる余裕 | 前のめりで相手の反応を伺う |
| 話題を振るとき | 自然な流れで切り出す。軽いトーン | 深刻な顔で「ちょっと話があるんだけど」 |
| 聞いた後 | 相手の言葉を受け止めてから返す。沈黙もOK | すぐ反論する。言い訳を引き出そうとする |
| 帰り際 | 楽しかった雰囲気で終わる。次の予定を自然に決める | 話が重いまま解散 |

### 話す順序（推奨フロー）

```
1. まず楽しく過ごす（食事・近況トーク）
   ↓
2. 自然な流れで切り出す
   ↓
3. 自分の気持ちを伝える（Issue 1 → Issue 2 の順）
   ↓
4. 「萌ちゃん的にはどう？」で相手に渡す ← ここがメイン
   ↓
5. 萌さんの話をじっくり聞く（遮らない・否定しない）
   ↓
6. 合意があれば自然にまとまる。なくても「聞けてよかった」で終わる
   ↓
7. 軽い話題に戻して楽しく終わる
```

## 話すこと / 話さないこと

### 話すこと（伝える内容）

**Issue 1 について — 切り出し例:**

> 「最近ちょっと思ってたんだけどさ、LINEが未読のまま長いと、
> 正直ちょっと寂しいなって思う時があって。
> 前に萌ちゃん自身が『既読つけとけば安心するでしょ』って言ってたじゃん？
> だから余計に、あれ？って思っちゃうんだよね。
> 忙しいのは分かってるんだけど。萌ちゃん的にはどんな感じ？」

**ポイント:**
- 「未読のまま」と具体的にする（既読スルーではなく未読スルーが本当に嫌なポイント）
- 萌さん自身の過去の発言を引用 → 責めではなく「あなたも分かってたよね」という事実ベース
- **「どんな感じ？」で終わる** → 解決策の提案ではなく、相手の考えを聞く形
- ピン留め等の具体策は、萌さんが「どうしたらいいかな」と聞いてきた場合にだけ出す

**Issue 2 について — 切り出し例:**

> 「あと、この前22日流れたじゃん？仕事だから全然いいんだけど、
> 俺としては、代わりの日いつにする？って言ってもらえると嬉しいなって思った。
> 萌ちゃん的にはあの辺どういう感覚だった？」

**ポイント:**
- 「全然いいんだけど」でキャンセル自体は受け入れ済みであることを示す
- 「嬉しいなって思った」= 過去形の感想。要求ではない
- **「どういう感覚だった？」で終わる** → 相手の事情や考えを引き出す
- 萌さん側にも理由があるかもしれない。それを聞くことが大事

### 話さないこと

| 内容 | 理由 |
|------|------|
| 「友達には返してるでしょ？」 | 比較・詮索は余裕がない印象になる |
| 「他に誰かいるの？」 | 疑いは絶対NG。信頼のなさが出る |
| 「もう少し頑張ってほしい」 | 上から目線。対等ではない |
| 不安だったエピソードの列挙 | 被害者ムーブ。重くなる |
| 「付き合ってまだ1ヶ月なのに」 | 期間を持ち出すと「こんなに早く面倒な人」感が出る |
| 自分がどれだけ考えたかの説明 | 「こんなに考えてた」は重い。サラッと伝える |
| 「冷めた？」「もう好きじゃない？」と直接聞く | 相手を追い詰める。不安の押しつけ。余裕ゼロ |
| 「好きって言ってほしい」 | 要求された愛情表現に価値はない。自発的に出てこそ意味がある |

## 自分への注意書き

### この会話の「成功」とは何か
- 自分の気持ちを正直に伝えられた → 成功
- 萌さんの考えを聞けた → 成功
- 二人の間で「お互いこう思ってたんだ」が共有できた → 成功

**萌さんが変わるかどうかは「成功」の定義に含まない。**

### 帰り道の自分チェック
- 言いたいことは言えたか？ → Yes なら OK
- 萌さんの話をちゃんと聞けたか？ → Yes なら OK
- 相手の返答に関わらず、自分はフラットでいられたか？ → Yes なら OK
- 次いつ会うか決められたら尚良い（でもこれも執着しない）
