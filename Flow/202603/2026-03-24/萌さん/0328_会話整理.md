## 戦略: 「余裕のある男」として話す

### 基本方針
- **責めない・詰めない**: 「なんで返さないの？」ではなく、自分の気持ちを伝える形にする
- **執着ゼロ**: 「返してくれないと困る」ではなく、「こうしてくれると嬉しい」で伝える
- **対等な関係**: お願いではなく、対等なパートナーとしてフラットに話す

### ノンバーバル（振る舞い）

| 場面 | OK | NG |
|------|----|----|
| 会った瞬間 | 笑顔で普通に「久しぶり」。ベタベタしない | 「やっと会えた…」と安堵感を全面に出す |
| 食事中 | リラックスした姿勢。背もたれに寄りかかる余裕 | 前のめりで相手の反応を伺う |
| 話題を振るとき | 自然な流れで切り出す。軽いトーン | 深刻な顔で「ちょっと話があるんだけど」 |
| 聞いた後 | 相手の言葉を受け止めてから返す。沈黙もOK | すぐ反論する。言い訳を引き出そうとする |
| 帰り際 | 楽しかった雰囲気で終わる。次の予定を自然に決める | 話が重いまま解散 |

### 話す順序（推奨フロー）

```
1. まず楽しく過ごす（食事・近況トーク）
   ↓
2. 自然な流れで切り出す
   ↓
3. 伝える（Issue 1 → Issue 2 の順）
   ↓
4. 相手の話を聞く
   ↓
5. 二人で「じゃあこうしよう」を決める
   ↓
6. 軽い話題に戻して楽しく終わる
```

## 話すこと / 話さないこと

### 話すこと（伝える内容）

**Issue 1 について — 切り出し例:**

> 「最近ちょっと思ってたんだけど、LINE返ってこない期間が長いと、
> さすがにちょっと寂しいなって思うことがあって。
> 忙しいのは分かってるんだけど、既読だけでもつけてくれると安心する。
> 前に忘れちゃうって言ってたけど、ピン留めとかしてくれたら嬉しいな。」

**ポイント:**
- 「寂しい」は伝えてOK（感情の共有であって、責めではない）
- 「忙しいのは分かってる」を先に言うことで、理解を示す
- 具体的な解決策（ピン留め）を提案 → 相手も行動しやすい

**Issue 2 について — 切り出し例:**

> 「あと、この前22日流れたじゃん？仕事だから全然いいんだけど、
> 代わりにいつにする？って言ってくれると嬉しかったな。
> 俺から毎回聞くのもなんかアレだしさ。」

**ポイント:**
- 「全然いいんだけど」でキャンセル自体は受け入れていることを示す
- 「嬉しかったな」= 過去形で軽く。要求ではなくリクエスト
- 「俺から毎回聞くのもなんかアレ」= ユーモアを交えて対等感を出す

### 話さないこと

| 内容 | 理由 |
|------|------|
| 「友達には返してるでしょ？」 | 比較・詮索は余裕がない印象になる |
| 「他に誰かいるの？」 | 疑いは絶対NG。信頼のなさが出る |
| 「もう少し頑張ってほしい」 | 上から目線。対等ではない |
| 不安だったエピソードの列挙 | 被害者ムーブ。重くなる |
| 「付き合ってまだ1ヶ月なのに」 | 期間を持ち出すと「こんなに早く面倒な人」感が出る |
| 自分がどれだけ考えたかの説明 | 「こんなに考えてた」は重い。サラッと伝える |
