## ⚠️ 1. 最初にチェック: そのURLで本当に申請できるか

以前発行されたURLには、どのタイプかによって「失効後の再交付には使えない」リスクがあります。

### 1.1 URLの出元・有効性の確認

以下を 5 分で確認してください（詳細は **INBOX.md の🔴緊急セクション** に質問を追記済み）:

1. **URLのドメインは何か？**
   - `net.kojinbango-card.go.jp` → J-LIS のオンライン申請サイト（本命。申請書ID付きURL）
   - `mynumbercard.point.soumu.go.jp` → マイナポイント用。**申請には使えない**
   - `myna.go.jp` → マイナポータル。**申請には使えない**
2. **URLはいつ発行されたものか？**
   - 前回カード作成時のQRコード/URLは、発行から数年経つと **申請書IDが失効している可能性あり**
3. **URLにアクセスして何の画面が出たか？**
   - 「メールアドレス登録」画面 → そのまま申請開始できる可能性が高い ✅
   - 「申請書ID入力を求める」画面 → IDを自分で入れる必要あり
   - 「すでに申請済です/受付終了」→ 使えないので新しい申請書が必要

### 1.2 判定結果別の次アクション

| 結果 | 次にやること |
|---|---|
| ✅ メールアドレス入力画面に進めた | → §3 のSTEPに従って申請実行 |
| ⚠️ 申請書ID入力を求められる | 申請書IDは旧カードに紐付くID。旧IDが有効なら入力して進む。不明なら ↓ |
| ❌ エラー / 受付終了画面 | ID付き申請書の再取得が必要。§5 のルートCへ |

> **判定が難しい場合**: 画面のスクショを共有してください。最適ルートを判断します。

## 2. 顔写真を用意する（申請開始前）

オンライン申請では最初に顔写真が必要です。スマホ撮影でOKですが、**規格を満たさないと差し戻し**になります。

### 2.1 写真の規格

| 項目 | 仕様 |
|---|---|
| ファイル形式 | **JPEG / JPG**（HEIC不可） |
| ファイルサイズ | 20KB〜7MB |
| ピクセル | 幅480〜6000 / 高さ480〜6000 |
| カラーモード | RGB（CMYK不可） |
| 撮影時期 | **6ヶ月以内** |
| アスペクト | 正面・無帽・無背景 |
| 加工 | 不可（美肌・フィルタ・補正すべて禁止） |

### 2.2 iPhone での撮り方（田中さん向け）

⚠️ **iPhoneの標準撮影は HEIC 形式**のためそのままだと弾かれます。以下のどちらかを実施:

**方法A: 設定変更**（推奨）
- 設定 > カメラ > フォーマット > **「互換性優先」** に変更
- 以降の撮影は JPEG で保存される

**方法B: 撮影後に変換**
- 撮影後に写真アプリで共有 → 「ファイルに保存」すると JPEG 化される環境もある
- 確実なのは「Image Converter」等のアプリで HEIC→JPEG 変換

### 2.3 撮影手順（5分）

1. **白い壁（無地）の前に立つ**（壁から30cm以上離れる）
2. スマホを棚・本・三脚等で固定し、**目の高さ** に合わせる
3. **窓際の自然光**で（フラッシュOFF）
4. セルフタイマー3〜10秒で撮影
5. 確認: 目がはっきり見える / 前髪が目にかかっていない / 眼鏡のフレームが目を隠していない / 影が顔に落ちていない / 背景に物が映り込んでいない
6. 服装: 襟付きでなくてOK（普通のTシャツ可）。帽子・サングラス・マスクは不可

### 2.4 NG例（差し戻しになる）

- 影がある / 背景が白くない / ピンボケ / 赤目 / 加工済み
- 眼鏡のフレームが目にかかっている / レンズに光が反射
- 前髪で目元が隠れている / 帽子 / マスク / サングラス
- HEIC形式のまま / CMYK / サイズが20KB未満または7MB超

## 3. オンライン申請の実行（step-by-step）

> 前提: URLが §1 で「メールアドレス入力画面に進める」と確認済み。

### STEP 1. サイトにアクセス、メールアドレス登録（3分）

1. スマホで発見した申請URLを開く
2. 画面の指示に従い **メールアドレスを登録**
3. 登録アドレスに「写真登録URL」が届く
4. メール内のURLをタップ

> 💡 定期メンテナンス時間: **第3土曜日 0:00〜8:00** はアクセス不可。それ以外は24h。

---

### STEP 2. 顔写真のアップロード（2分）

1. §2 で用意した JPEG ファイルを選択してアップロード
2. 画面の枠に顔を合わせる調整画面が出る場合あり（ガイドに従う）
3. エラーが出たら §2.1 の規格を再確認 → 撮り直し or 変換

---

### STEP 3. 申請情報の入力（3分）

以下の項目を入力:

| 項目 | 記入する内容 |
|---|---|
| 生年月日 | 西暦 / 和暦 |
| 電子証明書の発行希望 | **「希望する」**（両方: 署名用・利用者証明用） |
| 氏名の点字表記希望 | 通常「希望しない」 |
| 連絡先 | メールアドレス・電話番号 |

> 💡 **電子証明書は必ず「希望する」** を選ぶこと。希望しないと e-Tax・マイナポータル・コンビニ交付が使えないカードになる（田中さんの確定申告・BL-0014 用途からも必須）。

---

### STEP 4. 申請完了 → 確認メール受信（1分）

1. 「申請を受け付けました」画面を確認
2. メールで申請受付通知が届く → 保存しておく
3. **申請受付日から約1〜2ヶ月で文京区から交付通知書（はがき）が届く**

---

## 4. 申請後の流れ（受取まで）

### 4.1 交付通知書の到着（申請から約1〜2ヶ月後）

- 文京区 戸籍住民課から **青色の封筒のはがき（交付通知書）** が住民登録地に届く
- 書留（転送不可）で届く場合もある

### 4.2 受取予約（はがき到着後）

- 予約ページ: https://mynum.e-tumo.jp/mynumpo-bunkyo-u/reserve/inputMailto_initDisplay
- または電話: 03-5803-1666
- **受取は予約必須**

### 4.3 来庁して受取（30分程度）

- 場所: 文京シビックセンター 2階北側 戸籍住民課
- 住所: 東京都文京区春日1-16-21（後楽園駅 / 春日駅）
- 受付時間: 平日・第2日曜・第4土曜 9:00〜17:00（**水曜のみ19:00まで**）

**持参するもの:**

| 持ち物 | 備考 |
|---|---|
| 交付通知書（はがき） | 必須 |
| 期限切れの現カード | **必須**（回収される。紛失すると追加手数料） |
| 本人確認書類 | 顔写真付き1点（運転免許証・パスポート等）。旧カードは本人確認に使えない |
| 暗証番号メモ | 4桁数字 / 6〜16桁英数（覚えていれば不要、念のため） |

**窓口での流れ:**
1. 受付で交付通知書と本人確認書類を提示
2. 職員が新カードを準備
3. **暗証番号2種（4桁・6〜16桁）を入力** → カードに電子証明書が書き込まれる
4. 旧カードを回収、新カードを受領 → 完了

## 5. 万が一、発見したURLで申請できなかった場合

§1 の判定で「エラー / 受付終了」だった場合のフォールバック:

### ルートC-1: Logoフォームで ID付き申請書を取り寄せ（オンライン完結派）
1. 文京区のWebからLogoフォームへ移動（要確認）
2. 情報入力 → 数日〜1週間で住民登録地に申請書が郵送される
3. 到着した申請書のQRコードをスマホで読み取り → §3 と同じ流れで申請

### ルートC-2: 区役所窓口の「申請サポート」で当日対応
1. 03-5803-1666 で予約（平日 9:00〜16:30）
2. 窓口で ID付き申請書の発行〜写真撮影〜申請 をまとめて実施
3. 受取は §4 と同じ

## 6. 確定申告（BL-0014）への影響と対応

- 新カード到着まで **約1〜2ヶ月** かかるため、その間は e-Tax 不可
- BL-0014 は期限超過中なので、**書面提出に切替することを強く推奨**
- 新カード到着を待って e-Tax するのは遅延拡大のリスクが大きい
