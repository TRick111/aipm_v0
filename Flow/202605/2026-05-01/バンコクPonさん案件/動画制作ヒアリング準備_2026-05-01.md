## 2. 今後の調査の方向性（田中さん側で着手）

### 2.1 内製化 vs 外注の切り分け
- 動画はAI支援の中でも **学習コスト・運用コストが高め** な領域
  - 制作（編集・カット・字幕・テロップ・音声）: 業務側に時間が必要
  - 生成（AI動画モデル: Sora / Veo3 / Runway / Hailuo / Kling 等）: 品質はまだ用途次第
- **判断軸**:
  - 1本きり → 外注 or 田中さん同伴で町田さんが運用
  - 継続運用 → ポンさんチームに「型」を渡して内製化
  - 高品質ブランド動画 → 外部クリエイター推奨
  - 量産・ABテスト前提 → AI生成パイプラインの検討余地大

### 2.2 SNS投稿用動画と仮定した場合の調査領域
| 領域 | 調査ポイント | 優先度 |
|---|---|---|
| **タイ市場のSNS動画トレンド** | TikTok Thailand / IG Reels で美容院系がどう投稿しているか、視聴1秒目の引き、人気フォーマット | 高 |
| **競合美容院のSNS** | バンコクのトップ美容院 5-10店舗の投稿頻度・トーン・フック・再生数 | 高 |
| **既存リソースの棚卸し** | ポンさん側の写真・動画素材、スタッフ・モデル協力可否、撮影機材 | 高 |
| **AI動画生成ツールの現実解** | Runway Gen-4 / Veo3 / Sora / Kling / Hailuo / Pika の比較。タイ語ナレーション・実写顔キープの可否 | 中 |
| **AI画像→動画パイプライン** | Midjourney/Nano Banana → Runway / Veo3 で量産する型 | 中 |
| **編集の自動化** | CapCut / Descript / Opus Clip / Vrew(タイ語対応?) で人間の手を最小化 | 中 |
| **広告配信を伴う場合** | Meta広告のクリエイティブ最適化、Cookieless時代の指標、タイ国内予算感 | 配信先確定後 |
| **法務・著作・肖像権** | スタッフ/お客様の肖像権、AI生成顔の取り扱い、タイの広告表示規制 | 配信先確定後 |

### 2.3 田中さんの守備範囲を超える部分（=  協業 or 学習で埋める）
- **マーケティング戦略**: 町田さん or 外部マーケターの知見を借りる
- **撮影/編集の実務**: 既存の制作会社 / フリーランス活用前提で問題ない（田中さんが全部やる必要はない）
- **AI動画モデルの最新動向**: 田中さんが追うが、毎週変わるので「特定用途で当たりを引く」スタンスでよい
