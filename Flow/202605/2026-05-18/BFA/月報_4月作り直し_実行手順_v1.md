### 1.3 営業日基準への変換確認

```python
# datetime化 → 営業日変換
df['会計日時'] = pd.to_datetime(df['会計日'] + ' ' + df['会計時間'])

def to_business_date(dt):
    if dt.hour < 6:
        return (dt - pd.Timedelta(days=1)).date()
    return dt.date()

df['business_date'] = df['会計日時'].apply(to_business_date)
df['business_month'] = df['business_date'].apply(lambda d: d.strftime('%Y-%m'))

# 4月帰属の行数を確認
print('business_month=2026-04 件数:', (df['business_month']=='2026-04').sum())
print('会計日(暦日)=2026-04の件数:', (df['会計日'].str.startswith('2026/04')).sum())
# ↑ 数字が一致しないはず（深夜帯のズレ）
```

## 10. トラブルシューティング

| 症状 | 原因 | 対処 |
|---|---|---|
| business_month=2026-04 件数が会計日2026/04と大きく違う | 営業日変換が正しい証拠（4/30深夜＝4月、5/1早朝＝4月扱い） | 想定通り。エアレジ画面と件数一致を確認 |
