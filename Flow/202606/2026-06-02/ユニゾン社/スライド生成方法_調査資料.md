## 1. 調査対象: ZaraZhang/frontend-slides

### 概要

| 項目 | 内容 |

| リポジトリ | [zarazhangrui/frontend-slides](https://github.com/zarazhangrui/frontend-slides) |
| Stars | 約20,000 |
| 用途 | Claude Code向けスキル。アニメーション付きHTMLプレゼンテーションを生成 |
| 依存関係 | **ゼロ** — 単一HTMLファイル（CSS/JSインライン） |

### 主な機能

1. **新規プレゼンテーション作成**
   - テキストプロンプトからHTMLスライドを生成
   - 10種類のキュレーションされたデザインプリセット
   - ビジュアルスタイルのプレビュー比較機能

2. **PowerPoint変換**
   - 既存の `.pptx` ファイルをWebスライドショーに変換
   - テキスト・画像・ノートを抽出・保持

3. **デプロイ/エクスポート**
   - Vercelへの即時デプロイ対応
   - PDF出力対応
   - レスポンシブ 16:9 スケーリング

### インストール方法

**Claude Code Marketplace経由:**
```bash
/plugin marketplace add zarazhangrui/frontend-slides
/plugin install frontend-slides@frontend-slides
```

**手動インストール:**
```bash
git clone https://github.com/zarazhangrui/frontend-slides.git ~/.claude/skills/frontend-slides
```

### 使用方法

```
/frontend-slides

> "Create a pitch deck for my AI startup"
```

1. コンテンツ（スライド内容・メッセージ・画像）についてヒアリング
2. 印象（impressed? excited? calm?）についてヒアリング
3. 3つのビジュアルスタイルプレビューを生成・比較
4. 選択したスタイルでフルプレゼンテーションを作成
5. ブラウザで表示

## 4. 他の選択肢（比較用）

| ツール | 特徴 | 依存関係 |
|---|---|---|
| **frontend-slides** | Claude Codeスキル、アニメーション豊富 | なし |
| Reveal.js | 定番のHTMLプレゼンフレームワーク | npm |
| Slidev | Markdown → スライド、Vue.js製 | npm |
| Marp | Markdown → スライド、シンプル | npm |
| Google Slides API | プログラマティックなスライド生成 | API認証 |

### 結論

**frontend-slides を推奨**
- ゼロ依存で即導入可能
- Claude Codeとの統合がスムーズ
- デザイン品質が高い（Anti-AI-Slop）
- PPT変換機能があり既存資産を活用可能
