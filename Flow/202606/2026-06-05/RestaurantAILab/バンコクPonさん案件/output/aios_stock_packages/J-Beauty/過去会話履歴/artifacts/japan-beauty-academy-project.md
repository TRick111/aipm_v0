# **Japan Beauty Academy Project  
— 日本式美容をタイから広げる J-Beauty 美容学校構想 —**

このプロジェクトは、  
**「日本でも通用する美容師をタイで育てる」**  
ことを目的とした **J-Beauty 美容学校** の設立案です。

さらに、学校が成長するにつれて  
**国籍問わずタイで活躍するインフルエンサーとコラボし、  
J-Beauty の価値をASEAN全体へ発信していく仕組み**  
を融合させた総合プロジェクトです。

以下に、美容学校に必要な内容・設備・収益構造を  
すべて省略せずにまとめます。

────────────────────────  
# **① 美容学校の目的（3本柱）**

■ **① 日本の美容学校へ編入できる教育制度をつくる**  
→ タイで基礎単位を修了し、日本の美容学校へ編入できる仕組み。

■ **② 日本サロンで働ける技術・マナーを習得させる**  
→ 日本式カウンセリング、接客、技術基準を教育する。

■ **③ 日本でも通用する技術を身につける J-Beauty 技術教育**  
→ カラー・カット・撮影・トレンド・衛生管理すべてを学ぶ。

タイの美容学校との差別化は圧倒的で  
**“日本基準の美容師をタイで育てる学校”** という唯一の立ち位置になる。

────────────────────────  
# **② コース構造（例）**

■ **3ヶ月 J-Beauty Master Course（12,000〜30,000B）**  
・日本式カラー  
・ハイトーン基礎  
・作品撮影  
・日本式接客・仕上げ  
・モデル実習  

■ **6〜12ヶ月 日本編入コース（40,000〜120,000B）**  
・日本国家試験で必要な内容の基礎  
・日本語会話＋美容用語  
・日本式サロンワーク  
・日本学校との“編入制度付き”特別コース  

■ **Pon Kondo弟子コース（限定10名 / 60,000〜120,000B）**  
・Pon流カラー理論  
・作品撮影  
・現場同行  
・仕上がった作品をSNSで発信（希望者はPonが監修）  

────────────────────────  
# **③ カリキュラムの強み（日本基準 × ASEAN仕様）**

■ **日本の美容学校では必須の要素を網羅**  
・ブリーチ技術  
・カラー理論  
・ベーシックカット  
・観察・衛生・接客  
・国家試験内容（一部相当）

■ **ASEANで必要な“髪質特化ノウハウ”も教える**  
→ タイ・ベトナム・ラオス等の髪に合わせたブリーチ設計。

■ **撮影 × SNS発信も必須科目化**  
・作品撮影  
・編集基礎  
・モデル撮影  
・SNS運用  

→ 日本の美容学校でも強化されている分野を完全対応。

────────────────────────

→ **1つの学校で多層的に利益が取れる構造。**

────────────────────────  
# **⑧ インフルエンサーとのコラボ要素（美容学校との連動）**

ここでは **「美容学校を軸にしてインフルエンサーとコラボする形」** に絞る。

■ **① モデル撮影会でのコラボ**  
美容学生がインフルエンサーをモデルとして実習できる。  
→ インフルエンサーは無料でヘア施術＋撮影が受けられる  
→ 学校側はSNSで拡散される  

■ **② SNS課題の中でインフルエンサーと共演**  
学生が作品投稿するとき、インフルエンサーがタグ付け・ストーリーで紹介。  
→ 学生の成長速度が上がる。

■ **③ 先行体験イベントに美容学生が参加**  
新作コスメ・新技術のイベントに学生を同行させる。  
→ 若い美容師が最新の美容文化に触れられる。

■ **④ インフルエンサーが“特別講師”として参加**  
・SNS発信  
・撮影の心得  
・自分をブランド化する方法  

→ これが美容学生にとって圧倒的価値になる。

■ **⑤ Pon Kondo Academy × Influencer Project の合同イベント**  
学校の成果発表 × 撮影会 × Mini Fes  
→ 学校の知名度が爆発的に伸びる  
→ コラボによって入学希望者が増える  

インフルエンサーは“拡散力”  
美容学校は“教育力”  
この2つを組み合わせて成長させる。
