Ponさんの規模（複数店舗・スタッフ多い）だと、  
**「人の記憶に頼る方法」は絶対崩れます。**  
なのでポイントは **①自動で紐付く仕組み + ②誰が見ても分かるルール** にすることです。

実際に美容サロンで一番事故が少ないのはこの3つの方法です👇

---

# ① 顧客カルテと写真を紐づける（おすすめ）
一番ミスが少ない方法です。

**流れ**

1. 来店 → POSで顧客カルテ開く  
2. Before撮影  
3. After撮影  
4. そのままカルテにアップ  

つまり  

```
お客様名
↓
その人の写真
```

になるので  

**写真が混ざらない**

---

### 例えば

```
Customer
└ 山田さくら
   ├ before
   ├ after
   └ finished
```

この形にすると  
**他のお客さんと絶対混ざらない**

---

# ② 撮影 → LINEグループに送る（簡単）
サロンでよくやってる簡単な方法です。

グループ名  

```
BeforeAfter_投稿
```

投稿ルール  

```
店舗
スタイリスト
メニュー
before → after
```

例

```
YAMS
Hina
Bleach color

before
after
```

→ SNS担当が保存

---

# ③ Google Drive 自動フォルダ
もう少しちゃんとやるならこれ。

フォルダ構造

```
BeforeAfter

 ├ YAMS
 │   ├ 2026_03
 │   ├ 2026_04
 │
 ├ BELL
 │
 ├ MONDO
```

写真の名前

```
20260316_yams_hina_before
20260316_yams_hina_after
```

# ④ ミスを防ぐ最重要ルール
ここが一番重要です。

**撮影順番を固定**

```
① before
② after
③ 動画
```

必ずこの順番。

---

# ⑤ さらにミス減らす裏ワザ
これかなり効きます。

**写真の最初に手を写す**

beforeのとき

✋ 手を1枚撮る  

そのあと

before → after

すると

```
✋
before
after
```

になるので  
**区切りができて分かりやすい**
