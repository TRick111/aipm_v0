# 事業全体像俯瞰（PONさんのビジネスマップ）

## このファイルの目的

本ファイルは、PONさん（近藤Pon / Rios Innovation Co., Ltd. 社長）がバンコクを拠点に展開する多角的な事業群の全体像を俯瞰し、各事業領域の位置づけ、相互関係、そして戦略的な方向性を一望するための資料です。

PONさんの事業は、単なる美容室チェーンの運営にとどまりません。サービス業（美容室・専門店）、製造業（自社ブランド商品）、教育業（アカデミー）、そして文化産業（J-Beauty輸出）が複雑に絡み合いながら、「日本美容をアジアから世界へ」という一貫したビジョンのもとで有機的に連携しています。

会話履歴から抽出された4つのカテゴリ——美容室、美容専門店、自社ブランド、J-Beauty——を横断的に読み解くことで、個別事業の深掘りでは見えにくい全体構造、シナジー、そして横串の経営課題を浮き彫りにすることがこのファイルの目的です。

---

## 事業ポートフォリオ概観

PONさんの事業構造は、以下のような4層構造として理解できます。

```
【第1層：サービス基盤】
├─ 美容室事業（Rapi-rabi / YAMS / Cuu's / BELL / Rio Hair Design）
└─ 美容専門店事業（アイラッシュ / ネイルサロン / MONDO BEAUTY clinic）

【第2層：製品基盤】
└─ 自社ブランド事業（ヌリプラ / DOT / KINUJO提携 / 店販・EC）

【第3層：文化・教育基盤】
└─ J-Beauty事業（アカデミー / 政策提言 / イベント / インフルエンサー連携）

【横串】
├─ 人材マネジメント（給与制度 / 採用 / 育成 / 離職対策）
├─ ブランディング（日本品質の価値訴求 / 高単価ポジショニング）
├─ 海外展開（タイ→ベトナム→インドネシア / 日本市場との二軸経営）
└─ 不動産・インフラ（テナントビル運営 / 店舗内装設計）
```

### 事業間の関係性とシナジー構造

**垂直統合の軸**：美容室・専門店で培った現場知見が自社ブランド開発に活かされ、その製品が店販チャネルを通じて流通する。さらにアカデミーで育成した人材が各店舗に配置され、J-Beautyブランドを体現するサービスを提供する——という循環構造が設計されています。

**水平展開の軸**：タイで確立したモデルをベトナム、インドネシア、そして日本市場へと地理的に展開。各市場での学びが相互にフィードバックされ、事業モデル全体の精緻化が進む構造です。

**文化的統一性**：すべての事業が「J-Beauty（日本美容）」という文化的価値を核に据えており、単なる多角化ではなく、一貫したブランドストーリーのもとでの事業展開となっています。

## 各事業領域の解説

### 美容室事業

**現状**：バンコクを中心に複数ブランド・多店舗を展開する主力事業。カラー専門店「Rapi-rabi」は10店舗規模に成長し、フランチャイズ展開も視野に入っています。加えて、託児サポートの「YAMS hair & cafe」、個室設計の「Cuu's hair」、オトナ世代向けの「BELL otonagami」など、ターゲットセグメント別にブランドを分化させる戦略が取られています。

**主要な論点**：
- **人材マネジメントの最優先課題化**：給与制度設計（歩合・コミッション・家族手当）、離職対策、店長育成が経営課題の中核を占めています。特にRapi-rabiでは売上低迷と離職問題が顕在化し、ブランド再設計まで議論が発展しています。
- **地理的拡大**：ベトナム支店の運営、プーケット新店舗検討、さらにはインドネシア市場まで視野に入れた東南アジア全域への展開構想が進行中です。
- **日本市場との二軸経営**：Rio Hair Designを通じた日本市場での事業展開により、タイ・日本間の相互シナジー（人材交流、商品開発、ブランド発信）を追求しています。

**他事業との相互作用**：美容室は自社ブランド製品（ヌリプラ、DOT、KINUJO）の実証・販売チャネルであり、J-Beautyアカデミー卒業生の就職先でもあります。また、店舗運営で蓄積されたノウハウがアカデミーのカリキュラム開発に活かされています。

---

### 美容専門店事業

**現状**：美容室以外の専門的ビューティーサービスとして、アイラッシュ、ネイルサロン、そしてMONDO BEAUTY clinic（美容クリニック）を展開。いずれも「日本品質」を軸とした高単価・高付加価値ポジショニングを取っています。

**主要な論点**：
- **富裕層市場への明確なターゲティング**：2,000バーツ以上のネイルサービス、日本人管理者による品質管理体制など、価格競争ではなく価値訴求による差別化を徹底しています。
- **人材の質的担保**：2026年を見据えた日本人管理者投入計画、ボランティア企画を活用した求人戦略など、属人的でない品質管理システムの構築を志向しています。
- **テナントビルとの垂直統合**：プロンポンでの不動産事業と美容専門店事業を組み合わせ、テナント料収入とサロン収益の両方を確保する戦略が検討されています。

**他事業との相互作用**：専門店は自社ブランド製品（ヌリプラの施術用カラー剤、DOTのヘアケア製品）の実証・販売チャネルとして機能。また、MONDO BEAUTY clinicはJ-Beauty事業における「医療系美容」の旗艦店として、日本美容の信頼性を体現する役割を担います。

---

### 自社ブランド事業

**現状**：サービス業から製造業への垂直統合を志向し、プロ用カラー剤「ヌリプラ（NURI+）」、シャンプー・トリートメントの「DOT」、美容機器「KINUJO」との提携、そして店販・EC展開を推進しています。

**主要な論点**：
- **ヌリプラのブランド確立**：1年以上の検討を経てブランドアイデンティティを確立。海外展開を前提とした商標・ネーミング設計、タカラベルモントとの差別化戦略が練られています。
- **イノベーション志向**：KINUJOについては、528Hz・417Hzの周波数機能やEMS波形を組み込んだ次世代ドライヤー開発アイデアまで踏み込んでおり、既存製品を「超える」発想で新しい価値を生み出そうとしています。
- **店販の経営システム化**：売上目標を総売上の15-20%に設定し、コミッション制度・給与体系・目標設定を一体化させた戦略として店販を位置づけています。

**他事業との相互作用**：自社ブランド製品は美容室・専門店という自前の流通チャネルで実証・販売され、現場フィードバックが製品改良に活かされます。また、J-Beauty事業における「日本品質の証明」として、アジア市場への輸出商材の中核を担います。

---

### J-Beauty事業

**現状**：日本美容という文化的価値そのものをブランド化し、教育・人材育成・コミュニティ形成・ブランド構築を一体的に推進する包括的事業。PONさんはこの領域において「文化の編集長」として、日本美容の本質をアジア市場向けに翻訳・編集する役割を担おうとしています。

**主要な論点**：
- **アカデミー構想**：60人規模の美容学校設立に向けた費用計算、教室設計、運営システム構築が進行中。店舗勤務と学校教育を組み合わせたアプレンティスシップ制度により、教育と就職を一体化させた人材育成モデルを設計しています。
- **インフルエンサーネットワーク**：インドネシアのJohnny Andrean事例を参考に、金銭報酬ではなく特権・成長機会・仲間意識による持続可能なコミュニティ形成を志向しています。
- **ブランドポジショニング**：「安全で破綻のないデザイン」と「市場を変革する挑戦者としてのメッセージ性」のバランスが議論され、アジア美容業界における革命的存在としての差別化が模索されています。

**他事業との相互作用**：J-Beautyは他の3事業すべてを束ねる「上位概念」として機能。アカデミーは美容室・専門店への人材供給源であり、インフルエンサーネットワークは自社ブランドの認知拡大チャネルとなります。また、政策提言活動を通じて、タイ政府やJETROとの連携による制度的な競争優位性の構築も視野に入っています。

## 横串の課題（複数事業にまたがる論点）

### 1. 人材マネジメント体制の構築

会話履歴全体を通じて最も頻繁に言及される課題が人材です。美容室では給与制度設計・離職対策・店長育成、専門店では日本人管理者の投入計画、アカデミーでは教員確保と卒業生の配置——すべての事業において「人」が成長のボトルネックとなっています。

特に、タイの労働法遵守と日本的な品質基準の両立、現地スタッフのモチベーション維持と権限委譲のバランス、そして複数ブランド間での人材流動性確保が横断的な課題です。

### 2. 「日本品質」のブランド化と価格正当化

すべての事業が「日本クオリティ」を差別化の軸としていますが、その価値をタイ・アジア市場でどのように訴求し、高単価を正当化するかは共通の課題です。ヌリプラのネーミング戦略、ネイルサロンの2,000バーツ超えサービス設計、J-Beautyロゴの格調表現——いずれも「日本」という記号をいかに価値に転換するかという同一の問いに向き合っています。

### 3. 海外展開の組織体制

ベトナム支店、プーケット新店舗、インドネシア市場、そして日本市場と、地理的拡大の構想は多岐にわたりますが、それを支える組織体制（人材配置、現地パートナー選定、法人設立、品質管理の遠隔マネジメント）は共通の課題です。特に、バンコク本部のリソース制約のなかで、どの市場を優先し、どのような進出形態（直営・フランチャイズ・パートナーシップ）を選択するかは、全事業に影響する戦略的意思決定です。

### 4. 店販・EC・自社ブランドの流通戦略

自社ブランド製品を美容室・専門店で販売する「店販」モデルと、EC展開の両輪をどう組み合わせるかは、自社ブランド事業単体の課題ではなく、店舗スタッフのコミッション制度設計、顧客データ活用、そしてJ-Beautyブランドの認知拡大戦略にまで影響する横断的論点です。

### 5. 不動産・インフラとの垂直統合

プロンポンのテナントビル運営、店舗内装設計、設備保守といった不動産・インフラ関連の意思決定は、美容室・専門店の立地戦略、アカデミーの教室設計、そして自社ブランドの展示スペース確保と連動しています。
