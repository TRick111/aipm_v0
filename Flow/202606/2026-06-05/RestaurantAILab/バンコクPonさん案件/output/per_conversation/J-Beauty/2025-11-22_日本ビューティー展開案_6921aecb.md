# 日本ビューティー展開案

## ひとことサマリー
国の予算を活用し日本ビューティーをASEANに広げるプロジェクトの骨格設計。質問形式で7項目をPonさんから引き出し、商品物販×タイ中心ASEAN×一般女性×美容学校あり、2033年17兆円目標、来週プレゼン向け12ヶ月ロードマップに落とし込んだ。

## 論点と結論
- 国の予算取りに必要な「骨組み」5要素（目的定義／対象範囲／施策枠／KPI／ロードマップ） → SNS・教育・展示会・輸出支援・教育動画の5枠で予算を取りに行く設計
- Ponさんの優先軸 → 全分野対象／核は「商品・物販」／ターゲット国はタイ中心ASEAN／SNSターゲット一般ASEAN女性／美容学校を含める
- プロジェクトのゴール → 「日本ビューティー経済効果を2033年に17兆円」その達成にPonさん本人・周辺企業が貢献
- 期間 → 「最初は1年、毎月マイルストーン」型でプレゼン構成（12ヶ月ロードマップ）
- 韓国がなぜ日本で流行ったか → 政治面（文化輸出政策・ソフトパワー戦略）×エンタメ面（ビジュアル力／K-POP世界基準制作／恋愛感情訴求／プチプラ高品質コスメ）
- 韓国コスメへの日本の戦い方 → 「安心×信頼×成分透明性」「ASEAN専用処方の科学」「美容師×プロ技術」の3軸で韓国の逆張り
- タイ向けアンバサダー戦略 → 三浦孝太を「日本式メンズビューティー象徴」として起用検討、女性は新木優子、ファッション系は水原希子を3カテゴリ三刀流候補に
