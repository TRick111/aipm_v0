# J-Beauty戦略設計

## ひとことサマリー
PONさんは「J-BEAUTY ACADEMY / COSMETICS / TECH 詳細設計統合版」のドラフトを提示し、人件費の修正、Cosmetics/Techの締め文章の追加、K-Beautyとの差を実例とデータで再構成し、最終的に韓国のビューティー補助金実額（円ベース）と出典まで整理した。

## 論点と結論
- PONさんは Academy / Cosmetics / Tech の三事業を「信頼資産の蓄積」という1つのKPIに収束させる統合戦略書を提示し、アシスタントはこれを投資家・行政・メーカーに提出可能なレベルの完成度と評価したうえで人件費とOpExを再計算した。
- 人件費はタイ人6名＋リーダー1名追加を反映し、日本人500,000THB＋タイ人217,000THB＝月717,000THBへ修正し、月次OpExは家賃・外部講師月割・光熱・教材を含めて約1,138,000THBに着地させた。
- Cosmetics と Tech の章には「思想としての締め」が欠けているため、Cosmeticsを「診断・説明・提案による信頼消費への転換点」、Techを「使われ続ける日本技術の体験販売拠点」として位置づける追記文を作成した。
- JUNO HAIR比較を残しつつ、Academy単体で終わっていた「だからJ-BEAUTYはこうするべき」の章を、教育→Cosmetics→Tech→ASEAN→世界展開の連動構造で書き直し、三位一体モデルとして再定義した。
- PONさんは「よくある言葉を並べるだけの抽象表現」を拒否し、マーケッター視点でK-Beautyの過去事例（高頻度SKU投入・インフルエンサーギフト戦略・K-POPとの連動）を出典付きで再構成するよう指示し、アシスタントは引用付きで書き直した。
- 韓国政府のK-Beauty支援は単一巨大予算ではなく、K-Beauty投資ファンド（約44億円）、輸出バウチャー（約62億円）、化粧品産業1兆ウォン支援策（約1,100億円）、KOTRA全産業枠（約1兆円）が積み上がる構造であり、結果として2024年の化粧品輸出は約1兆5,300億円に到達したと整理した。
- まとめの数字には韓国中小ベンチャー企業部、Invest Korea、KoreaWave、Korea.netという一次・準一次情報を出典として明記し、資料転載用の一文も作成した。

## 決定事項・アクション
- 人件費は日本人5名＋タイ人7名構成で月717,000THB、月次OpEx約1,138,000THBに確定する。
- Cosmetics と Tech の章末まとめ文と統合戦略最終章は、アシスタント提供のドラフトをそのまま差し替える。
- 韓国補助金は円ベース表記に統一し、出典URL付きで資料に転載する。
- 次のステップ候補（ピッチ1枚資料／カリキュラム商品化／J-Beauty Standard草案／ASEANロードマップ／日本側の予算対比設計）からどれを着手するかをPONさんが選ぶ。
