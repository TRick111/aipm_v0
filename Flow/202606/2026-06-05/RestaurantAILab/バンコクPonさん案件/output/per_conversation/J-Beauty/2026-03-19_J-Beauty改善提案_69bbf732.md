# J-Beauty改善提案

## ひとことサマリー
野田康司氏（J-Beauty海外展開推進協会理事）の第6回J-Beauty産業研究会向け国家戦略提言資料を読み込み、「お願い資料」から「投資案件」に変える改造方針を提示。Ponモデル（CUU/ラピラピの現場成功）追加・"体験産業シフト"・5つの証明ポイントの差し替えを提案した。

## 論点と結論
- 原資料の本質 → 「J-Beautyは負けた理由が構造的で企業努力では覆せない」証明資料。直近20年でK=18.64倍/C=50倍/J=2.42倍の市場推移
- 資料の強み → ①市場の現実数字 ②国家vs個人の構図 ③50-90%実弾支援＋実務部隊＋教育輸出の具体ソリューション
- 弱点 → "危機止まり"で勝ち筋の絵が弱い。"お願い資料"になっている
- 改造方針① ストーリー変換 → ①Jはまだ勝てる市場(証明)②勝ち筋を国家でスケール③だから投資すべき
- 改造方針② "Ponモデル(現場成功)"1枚追加 → CUU/ラピラピの単価・再来率・教育LTVを再現可能モデルとして提示
- 改造方針③ 競争軸を体験産業に → "Jはプロダクトでは負ける、でも"体験"では圧勝できる"
- 改造方針④ 金額の説得力 → 14兆円抽象論+1店舗売上・1人教育LTV・1国展開のミクロ数字
- 改造方針⑤ ラスト一撃 → "支援してください"→"今やらないと終わる"の恐怖換言＋"これは投資です"宣言
- 「現地の声」スライド再設計 → "困っている"羅列から、①日本式技術への需要②教育需要③日本人スタイリストの価値④制度資金の壁の4軸構造化、結論「J-Beautyは"売れない"のではなく"供給体制が整っていないだけ"」
- PONさんっぽい言葉版（パンチライン）→ 「上手い美容師がいないんじゃなくて、売れる美容師がいない」「K-Beautyはマーケで勝った。J-Beautyは体験で勝てる」
- 5つの証明ポイント → ①需要は数字で(352店舗参加→購入率) ②成功モデルは自分の数字で ③教育は収益で ④K/Cとの差別化は比較データで ⑤支援効果はリターンで

## 決定事項・アクション
- J-Beauty提言資料に「Ponモデル」「体験産業シフト」「具体数字」3枚を追加して進化版を作成
- 「現地の声」スライドを「市場の証明」スライドに作り替え（4軸構造化＋結論「供給体制が整っていないだけ」）
- 最終ラストページを「投資すべき理由」（リターン/再現性/タイミング）の構成に差し替え
- Ponさんの単価/再来率/教育LTV/店舗売上の数字を整理してミクロデータをスライド化
