# JBT予算と戦略

## ひとことサマリー
JBT産業研究会向けに、Ponさんが「現地経営者の一次情報」として何を主張するかを整理。海外展開する日系美容事業者の資金調達難・言語壁・人材育成不全・教育単体限界・コンテンツ弱さの5論点に集約し、AI/POSプロジェクトの位置付けと補助金の流れも整理した。

## 論点と結論
- 市場規模を10兆円規模に膨らませる必要性 → パイをでかくしないと予算が取れない。原料協会の1兆円を加えて6兆超を狙う。10倍拡大の根拠は韓国18.64倍/中国50倍
- 2026–2028で現実的に通る政策 → 「実弾支援」と「実行ユニット整備」「認証制度」「教育×商品の輸出実証」がリアル。アカデミーがいきなり満額で来ることはないと予想
- Ponさん名義で出すべき要望5本 → ①日系事業者への融資/補助、②現地人材教育・認証・翻訳支援、③教育×商品×機器一体型輸出モデル、④講師ビザ/短期招聘/通訳費補助、⑤J-Beauty発信・アイコン化支援
- 現地での資金調達 → 外国人経営者はフルキャッシュで出店、借りられても年利12–14%、給与改善のハンドリングが効かない
- 言語壁 → N3層は残るがN1は大手に流れる、広告かけたら捕まる、ローカルで採用しても通訳コストが重い
- ラピラビの取り組み → タイ人スタッフ100%で月150万バーツ売る店長を実現。ただし他社が真似るには資金力と覚悟が要る
- J-Beautyのコンテンツ弱さ → フランスでKビューティーだらけ、Jビューティーは2本だけ。アジア発のアイコン（ブラックピンク的存在）がない
- AI/POS/在庫案件 → 予約掘り起こし（LINE登録15万人）/POS×ビフォーアフター×AI解析/スマホ棚卸し3層。「6月以降に具体化」「制度設計に応じて領域選定」と伝える
- お金の流れ3パターン → A: JBA経由が現実的、B: 直接公募はハードル高、C: JETRO経由展示会枠
- コスモプロフ補足 → 紙ピタ（日本人社長）から声、ヒストリック(ネイル)/コーセー/ロア/マーブの動き。"クレイジー"なところが必要

## 決定事項・アクション
- 月曜午後の入稿に向けJBT産業研究会資料を最終化
- 「ゲリラ戦」を「個社別の消耗戦／単独展開」に変更
- 4月第1週に7回目の研究会あり、その後取りまとめ
- 6月までAI案件は具体化しない（制度待ち）
- 紙ピタには「資料あれば確認」で既読無視継続
- 4/1コスモプロフのスペース調整（プレミアム/コア/テストマーケット）
- 7月セントラルワールド企画でJBA経由の実務を回す方針
- 14兆円ロードマップは2026–2028の中身を厚くする
