─────────────────────────  
# ⭐ Pon式 「無料講習で意識低い人を自然にふるいにかける方法」

これは“強制”ではなく、  
**自然と本気だけが残る仕組み**なので、  
学校の雰囲気が守られます。

─────────────────────────  
# 【① 参加フォームの質問でふるいにかける（入口）】

まず最初のフィルターはここ。

Googleフォームに最低限入れる質問：

✔ 美容歴  
✔ 今の悩み  
✔ なぜこの講習に参加したいのか？（自由記述）  
✔ 日本で美容師になりたいか？  
✔ SNSアカウント（本人確認兼ねる）  

ポイント  
👉 **“熱量の低い人ほど答えが適当”**  
👉 参加希望が30名以上のときは、  
　入力内容が薄い人から落とせばいい。

→ 無料講習でも「応募段階で意識の差が見える」

─────────────────────────  
# 【② 当日に「必ず守るルール」を提示して、守れる人だけ残る】

例：  
・遅刻は15分まで  
・講習中の私語NG  
・来たら必ず名前を書いてCHECK IN  
・SNSに1つ投稿すること（強制ではないが推奨）  

これを事前にLINEで送っておくだけで  
**意識が低い人は自動的に来なくなる。**

→ 来ても空気を読んで居心地悪くなるため、次回は来ない。

─────────────────────────  
# 【③ 無料講習は“立ち見 or 簡易イス”にする（意識の低い人ほど座りたがる）】

無料講習だから“高級感”は不要。

・簡易椅子  
・床座り  
・立ち見  
・席は早い者勝ち  

にすると、

👉 本気の子：気にせず来る  
👉 意識の低い子：気まずくて来なくなる  

※ 有料講座はしっかりした環境にすれば問題なし。

─────────────────────────  
# 【④ 名簿＆出席カードで“継続の本気度”を可視化する】

来る人全員に：

◎ 出席カード（スタンプカード）  
◎ 3回来たら「Ponによる技術評価」  
◎ 5回来たら「特別受講枠」  

こうすると

・本気の人 → 毎月来る  
・意識低い人 → 自然に脱落  

→ **努力が見える人だけが学校の核になる**

これは学校を成功させる“文化作り”そのもの。

─────────────────────────  
# 【⑤ 無料講習では「何もしない人の居場所がなくなる」空気を作る】

※ 意識の低い人を排除する最も効果的な方法。

やり方は簡単：

・講習中に質問タイムを入れる  
・グループで2〜3分話させる  
・簡単なワークを入れる（配色判断など）

👉 本気の子：積極的に参加  
👉 意識の低い子：動けなくて居心地が悪い  
→ 2回目以降来なくなる

あなたが攻撃的になる必要はゼロで、  
**自然と意識の低い人が消える。**

─────────────────────────  
# 【⑥ 無料講習後の“次のステップ”でふるいが完成する】

講習が終わったら：

#### 「次回の講習は、今日の内容の続きです」  
#### 「希望者のみ、Ponの技術チェックがあります」

こう伝えるだけで

👉 本気の人：残る  
👉 意識低い人：自主的に離脱

※ 必要以上に厳しくしないほうが逆に効果が高い。
