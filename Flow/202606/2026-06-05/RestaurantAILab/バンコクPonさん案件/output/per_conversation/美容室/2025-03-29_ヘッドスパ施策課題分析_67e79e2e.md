# ヘッドスパ施策課題分析

## ひとことサマリー
5月開始の rapirabi ヘッドスパ導入計画のリスク洗い出しから、ミャンマー地震の余震、RISA店舗の4月売上目標分析、エステ広告予算・プロモ期間・価格・三つ折りパンフサイズまで、当日の運営課題を横断で相談した実務会話。

## 論点と結論
- ヘッドスパ導入の想定リスク → ①予約集中によるキャパ超過（8面）、②インフルエンサー投稿の効果分散、③カウンセリング／通訳の品質バラつき、④日本人客への「プロフェッショナル感」不足。各対策として予約枠細分化、投稿スケジュール分散、ハッシュタグ #rapirabiヘッドスパ 統一など。
- スタッフ強化 → サイモンのカウンセリング／アフターカウンセリングをフローチャート化、ロールプレイ実施。りんちゃんの日本語表現はプロ感のあるスクリプトでトレーニング。
- 2025年3月28日ミャンマーM8.2の余震 → タイ気象局報告で27回観測、最大M7.1。今後1〜2週間は要警戒、高層ビル特に注意。
- 一般的な余震パターン → 本震直後数日が最多、1〜2週間頻発、数ヶ月〜1年続くこともある。本震より1〜2マグニチュード小さいのが通常。
- RISA店舗3月実績の問題 → 売上650,000B目標→493,730B（76%）、来店92→67名、店販30,000B→1,990B、次回予約率51%。原因は価格差（SERUMが日本19,800円より高い5,000B）、来店スパンをお客様任せ、SNS素材不足。
- 改善案 → SERUM価格を日本水準に合わせる／付加価値で説明、新規3週間スパン提案、2日前リマインド継続、KozueさんIGを参考にSNS素材作り。
- インスタ広告テスト予算（3万円エステ） → 7日間で15,000〜30,000円、1日2,000〜4,000円が理想。ターゲット絞り込み必須。
- プロモ期間 → 高単価は1ヶ月推奨。即決購入が難しいため認知の蓄積期間が必要。月初〜月中（1日〜15日）が効果出やすい。
- 7,800円から下げる語呂のいい価格 → 7,000円（ラッキーセブン）、6,800円（ロッパチ）、6,500円、5,800円。
- 三つ折りパンフ縦95×横125mm相当 → 完全一致はないが、B7（88×125）、A6（105×148）、DL系、長3が近い。
