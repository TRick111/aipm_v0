# タイでカラー剤販売

## ひとことサマリー
ラピラビオーナーがミルボンをベンチマークに70色OEMのプロ向けヘアカラー剤をタイで開発・販売する計画について、市場規模/競合/差別化/連携/類似事例/リスクの観点で勝算評価を実施。「成長機会大、自社サロンを軸にした段階的拡大で十分勝算あり」と結論。

## 論点と結論
- タイのヘアカラー市場 → 2018年で37.6億バーツ、年平均7.7%成長（ヘアケアで最大）。プロ向け業務用市場は53.7億バーツ（年6.6%成長）。低刺激・ヴィーガン・アンモニアフリーへのニーズ、EC普及も追い風
- ミルボンの強み → 2012年タイ・ラヨーンに国外初工場、アジア人毛髪研究のCMADK技術、Ordeveシリーズ120色以上、輸出シェア23%（1位、ホーユー21%が2位）、教育・サポート体制
- 競合 → ロレアル（fufu導入）、シュワルツコフ、ウェラ、ホーユー、資生堂プロフェッショナル
- 自社OEMのポジショニング → 「高品質×手頃な価格」、現地生産・直販で中間マージンを削減、「ローカル発のプロ向けカラー剤」ブランドストーリー
- 差別化4軸 → ①70色＋トレンド機動力 ②低刺激・高ダメージケア（ボンド系/オーガニックオイル）③アフターカラーケア提案（色持ちシャンプー）④パーソナルサポート（小ロット/講習無料）
- 自社サロン活用 → フラッグシップ、年間5万人施術データ、オーナーアンバサダー（ガイ・タン#mydentityのように）、ラピラビ認定カラーサロン
- ディーラー連携戦略 → バンコク都心・日系サロン密集エリアから攻略、エリア戦略、キーパーソン口説き
- 成功例 → ガイ・タン#mydentity（Henkel製造、SNS発信で世界拡大）、COLORIS（資生ケミカルOEM、1.5年で会員3万）、Lolane（タイローカル成功）
- 失敗例の課題 → 流通網の弱さ、ブランド認知不足、色数SKUによる在庫負担、品質トラブル
- リスクと対策 → ①競合優位への挑戦→ニッチ攻略 ②品質→OEM選定+PL保険 ③在庫→売れ筋集中＋受注生産 ④流通失敗→直販で採算 ⑤人材→外部委託 ⑥認知→教育・アンバサダー
- KPI提言 → 1年目で主要5色リピート率○%/クレ0、3年目でタイ国内シェア○%、2年目で単月黒字化

## 決定事項・アクション
- 段階的拡大戦略：①直営+モデルサロン→②首都圏ディーラー→③全国+ECの3段階
- 重点ターゲット：日本人経営サロン＋若年トレンド層＋白髪世代
- OEM工場は実績ある複数社と比較、サロン現場テスト後にローンチ
- PL保険加入、タイFDA規制チェック
- オーナーアンバサダー戦略（メディア、SNS）を強化
