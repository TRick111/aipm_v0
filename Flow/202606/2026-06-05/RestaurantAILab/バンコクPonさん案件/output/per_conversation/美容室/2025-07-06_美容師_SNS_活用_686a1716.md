# 美容師 SNS 活用

## ひとことサマリー
タイ在住美容師としてアート×ファッション系のInstagramで認知を広げ、日本のテレビ出演までを目指す戦略を整理。続けて、共有された LIQUID/YAKUZA/STATUS/DARK CLUBBING等のYouTubeミックスサムネ風ビジュアル別の撮影ディレクション案と、LIQUID Vol.2 撮影に使うピンクリキッドのレシピまで踏み込んだ。

## 論点と結論
- Instagramでアート・ファッションのフォロワーを伸ばす型 → ①テーマ統一 ②ハッシュタグ（10個前後、ニッチ＋人気） ③朝8-9時／夜21-22時に投稿 ④Reels/ストーリーで多形態運用 ⑤他SNS連携
- 海外日本人インフルエンサー成功事例 → LA在住美容師の毎日リール／バンコク在住Syoko（日本語＋タイ語）／Marikoのタイ生活11年発信
- 国内アート系成功 → 山之内すずさん（インスタ写真でスカウト）、駒木根葵汰さん（イケメン高校生→俳優デビュー）
- 美容師でテレビ出演した事例 → 米澤香央里さん（作品継続投稿→写真展→MV/ファッションショー/TV）
- 日本のメディアが注目する要素 → ①異色の組み合わせ・肩書き ②共感ストーリー ③トレンド性／フォロワー数 ④人柄・キャラ ⑤時事性・社会接点
- フォロワーから出演に繋ぐ目安 → 1万人（マイクロインフルエンサー）／10万人（一般メディア）
- 売り込みルート → 番組視聴者参加コーナー応募／PRマネジメント事務所（C Channel／VAZ等）所属／プレスリリース／タイメディアでの逆輸入
- タイというロケーションの活かし方 → 異国情緒な写真、多文化ミックススタイル、ジオタグ・現地クリエイターとのコラボ
- LIQUID Vol.2撮影のピンクリキッド → ①グリッター＋透明ジェル＋着色料 ②コーンシロップ＋食紅自作 ③SFXメイク用（Kryolan/Mehron/MAC）
- 各サムネ系のディレクション → LIQUID="濡れ髪＋グリッター涙＋赤グラデ"、YAKUZA="背中ヘアアート＋和×モード"、STATUS="ラグジュアリー×ストレート＋光沢小道具"、PRIVÉ/DIVINE="角・光輪ヘアオブジェ＋ハーネス"、BLOOD RAVE="赤×黒マーブル＋牙メイク"

## 決定事項・アクション
- まずはInstagramを"主役はヘア＋顔・目力"の世界観で統一して継続発信
- 第1作目として LIQUID Vol.1 "Red Desire"（濡れ前髪＋グリッター涙＋赤背景）を撮影予定
- ピンクリキッドはグリセリンジェル＋食紅＋微粒子グリッターの自作レシピで試作
- フォロワー数を段階的に伸ばし、マイクロ→事務所所属→日本メディア露出のステップを意識
