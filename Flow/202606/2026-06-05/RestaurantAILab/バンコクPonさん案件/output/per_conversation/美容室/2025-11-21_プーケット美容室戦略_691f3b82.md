# プーケット美容室戦略

## ひとことサマリー
プーケット・オールドタウンに70㎡5席で美容室を出店すべきかをラピラビ既存運営の延長で検討し、インバウンド依存設計／狙うべき3国／Ratsada通り物件の視察チェックリスト10項目まで現地視察直前に詰めた会話。

## 論点と結論
- 単価1,700Bは地元客で成立しない → ローカル相場は400〜800B。インバウンド比率60〜90%が必須
- 国籍別の現場感（Pon現場意見）→ インド人増加／中国人減少／白人横ばい。ただし美容室を使うインド人は20-30代女性ブロー・巻き髪需要
- 狙うべき3国 → ①韓国（最優先・日韓ミックス相性最高・Reels/TikTok集客）②欧米（Google Map最強・海ダメージケア・高単価）③インド（Reels爆伸び・ブロー需要・派手スタイル）。中国団体・日本人・マレーシア・中東は外す
- ロシア／カザフ → 同率3位の新興客層。長期滞在で美容室利用率高い
- SNS動線 → 韓国: Instagram Reels／TikTok／ネイバー検索／ハングル投稿。欧米: Google Map (90%)／Instagramリール／TripAdvisor。インド: Reels／Facebook
- 出店タイミング → 今出すと6-9月ローシーズンで赤字リスク。2025年10月〜2026年2月のハイシーズン前出店が安全
- 70㎡5席の評価 → 黄金サイズ。世界観確保＋撮影可能レイアウト。回転低いインバウンド客層に適正
- 収支イメージ → 月固定費13万B〜、月売上27万〜40万B、ハイシーズン利益7-15万B、ローシーズン0〜赤字
- 売れるメニュー → 海ダメージ補修トリートメント (1,500-2,200B)／スタイリング (900-1,300B)／フォト向けセット (1,200-1,600B)／カラーリタッチ (1,500-1,900B)
- 視察チェックリスト10項目 → 通行量（昼/夕）／物件前10分滞留率／徒歩5分以内ホテルマッピング／競合美容室・スパ／Google Map評価／夜の治安・照明／物件の天井高・採光／スタッフ採用難易度／国籍観察／観光客直接インタビュー

## 決定事項・アクション
- 場所はラッサダー通り（オールドタウン）。立地はOK、ただし“場当たり的に出店するとリスク”
- 視察は Day1: 通行量＋ホテル＋競合＋Google Map＋夜の通行量／Day2: 昼動線再確認＋物件採光＋家賃交渉＋採用難易度の2日構成で進める
- 写真（外観／内装／通行量）数枚送ってもらえれば30分以内で経営判断レベルにまとめる、と提案

## 言及された人物・店舗・商品
- Pon Kondo — ラピラビ10店舗運営オーナー。本人がプーケット現地視察中
- ラピラビ（バンコク） — 既存ブランド。プーケットへの延長店舗を検討
- Ratsada Rd（ラッサダー通り）／オールドタウン／パトン — 候補エリア
- Dusit / Anantara / Hyatt — 高級ホテル投資ラッシュの言及
