# カラー名の付け方

## ひとことサマリー
自社カラー剤「塗りプラス／NURI＋」のネーミング設計を、記号系（A14/N14）と世界観系（ビーチ/サファリ）の比較から「ハイブリッド型（番号×状態名）」に着地。Shadow/Bare/Clear/Calm/Veil/Flush系18色を設計し、Comfort Natural/Warmの対概念まで言語化した。途中、ベトナム空港のディプロマティックレーン誤入りも対応。

## 論点と結論
- 記号系vs世界観系vsハイブリッド → 記号は導入・教育に強いが差別化弱、世界観は覚えやすいが現場混乱。10年伸びたブランドは全て「両立型」
- 塗りプラスの最適解 → 正式EL14/A14＋通称Beach/Safariのハイブリッド。色名を書かなくても出方が想像できる「状態・現象」名が思想と合う
- 蒸気思想抜きの状態名 → Ash=Shadow, Natural=Bare/Clear, Matte=Calm, Purple=Veil, Pink=Flush
- 3タイプ比較
  - ①世界観だけ：覚える/SNS強い／現場運用重い・SKU管理崩壊
  - ②Ash/Natural系：プロ即理解・教育速い／差別化困難
  - ③ビーチ/サファリ系：覚える・SNS強い／教育コスト・新規ブランドで信頼弱
- 推奨フォーマット → ［ライン記号］＋［レベル番号］＋［ニュアンス記号］＋（通称ネーム）。例: NURI+ A14 Shadow
- 今やるべきTop3 → ①番号と記号の辞書（14/A/N/M/Pの意味）を1枚早見表に ②通称ネームの世界観を1系統に統一 ③ローンチは5-12色に絞る
- NURI+ 18色ネーミング → クール補正系（Shadow/Mute/Dust/Veil/Still/Fogless）/ベース系（Bare/Clear/Soft/Pure）/寒色〜ニュートラル系（Calm/Neutral/Balance）/血色系（Flush/Blush/Warm）/ニュアンス系（Aura/Bloom）
- 売上7割の核 → Shadow / Bare / Clear / Flush の4色
- Comfort Natural（赤みなし茶色）の対 → Comfort Warm（赤みあり茶色）。「ナチュラル vs ウォーム」で色相が一瞬で伝わる
- Comfort Naturalの意味 → 直訳「心地よく自然な」、「安全で失敗しにくい茶色」、色そのものより仕上がりの体感名
- ベトナム空港ディプロマティックレーン → 外交官/公用パスポート専用、一般旅券はNG。優先入出国は「表示」ではなく「事前登録or アテンド」（Fast Track $30-60で15-45分短縮）

## 決定事項・アクション
- ハイブリッド型「番号×状態名」で設計
- NURI+の18色ネーミング案を採用候補に
- 番号意味の早見表を作る
- 通称ネームは「状態・質感」系（Shadow/Clear/Calm/Veil/Flush）に統一
- ローンチは「Shadow/Bare/Clear/Flush」の4軸を核に
- ブラウン系は Comfort Natural / Comfort Warm の2本立て
- 3ライン（CORE/CONTROL/NUANCE）への分割を次の一手で検討
