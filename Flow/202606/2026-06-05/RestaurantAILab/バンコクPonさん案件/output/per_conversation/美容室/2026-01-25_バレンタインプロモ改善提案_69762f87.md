# バレンタインプロモ改善提案

## ひとことサマリー
12年続く美容室のバレンタイン「ペア割10%OFF＋ヘッドスパFree」案を起点に、土日稼働＋利益保護の観点から割引主義から脱却し、「来る理由づけ＝診断×体験×継続」の方向へ転換。店長への伝え方文章まで作成、最終的に「ペアガチャポン」案の格上げ＋"2月の髪と頭皮リセット月間"を結論として提示した。

## 論点と結論
- 原案「ペア割10%OFF＋ヘッドスパFree」評価 → 7.5-8/10。ペア条件が広い・特典の割引×体験バランス・現実的条件が良い
- 土日埋まる店での割引リスク → 本来フルプライスで埋まる枠を割引で消費。利益圧縮装置になる懸念
- 「平日限定」が寒く見える問題 → 強い店ほど"混んでないんだな"と見透かされるフェーズ。代案として「予約枠限定／数量制限／告知しすぎない」
- バレンタインの本質 → "恋愛イベント"ではなく"一緒に来る言い訳イベント"。割引で理由を作るから違和感
- マーケター推奨3型 → ①口実設計型（割引なし＋体験付加）②数量制限型（曜日でなく数で切る）③関係性強化型（次回予約＋紹介で動かす）
- 2026年バレンタインデータ → プレゼント予定者40.3%（減少傾向）、相手も恋人/パートナー減。"自分用"と家族・友人に広げると勝ち筋
- "髪質改善"を今っぽく見せる言い方 → "技術名"より"原因特定→施術選定→自宅再現の設計力"を売る
- 店長への伝え方フレーム → ①本気を評価 ②視点のズレとして共有（フェーズが違う）③問いを投げる ④代替案を一緒に考える形
- 最終結論（しっくりこない場合） → バレンタインを主役にしない。「2月の髪と頭皮リセット月間」として静かにやる
- 「ペアガチャポン」改良案評価 → コンセプトは◎、ガチャ中身を「200B OFF/チョコ」から「ミニスパ／質感アップ／頭皮チェック／次回ケアアップグレード」へ格上げ
- 土日価値保護の決まり文句 →「特典は丁寧に行うため、予約枠限定となります」

## 決定事項・アクション
- バレンタインを"割引イベント"から"理由づけ・体験イベント"へ転換
- ペアガチャポン案を採用しつつ中身を「美容体験寄り」に格上げ
- 土日制約は「予約枠限定」表現で表現（"平日限定"は使わない）
- 店長にはマーケ目線を「フェーズの違い」として伝える文章を共有
- 静か版/全体配信版のLINE文章を用意（既存VIP多めなら静か版を選択）
