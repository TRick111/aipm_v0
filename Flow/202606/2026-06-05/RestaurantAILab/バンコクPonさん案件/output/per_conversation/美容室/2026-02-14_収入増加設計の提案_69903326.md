# 収入増加設計の提案

## ひとことサマリー
ヤムズ山田店長（54歳・手取り11〜12万B）から「給料が足りない」と相談された PON が、固定給を上げずに収入を増やす構造（疑似FC→完全実装実験機）を設計し、当日の面談スクリプトまで詰めた会話。

## 論点と結論
- 「給料が足りない」への対応 → 固定給アップは絶対NG。固定費膨張・他店長への波及・利益圧迫リスクが高すぎる。「足りない」の裏は"将来の見通しが見えない"心理。
- 現状給与構造の数字検証 → 基本給70,000B＋総売上7%＝想定154,000Bだが実支給13〜14万。総売上が想定より低くロスしている。山田個人売上は約20万Bしかない。
- 収入を増やす4つの選択肢 → ①店長歩合強化、②カット講習の事業化（J-Beautyアカデミー講師化、1回5,000B×10人）、③技術ディプロマ販売（ジャパンレイヤー・くせ毛補正等）、④FC候補育成ポジション。
- フランチャイズ化の検討 → ①マネジメントFC（疑似オーナー型）と②完全FC売却型を段階式デュアル設計で。フェーズ1は1年間の疑似FC、利益25%還元で18〜20万レンジへ。
- ロイヤリティと加盟金 → 売上ベース5%＋広告1%が現実ライン（利益操作回避のため売上ベース）。加盟金は既存店FC転換なら80〜150万B、推奨100万B。
- 「実験機」化への転換 → 校舎モデルではなく完全実装実験機（Full Implementation Lab）へ。固定給5万B＋営業利益30%還元モデル（％が変わったのはフェーズが「半経営者」から「本気のオーナー思考」へ移ったため）。
- 山田の適性問題 → SNS不可・広告作れない・54歳で「学ぶ意欲はあるが覚えられない」。さらに17時完全退勤・休日家族優先。これはFCオーナー型・実験機責任者型ではなく「技術資産（技術統括・教育専任）」型。
- 面談トーン設計 → 今日は「白黒つける日」ではなく「構造を提示する日」。リスペクト明示→会社方向性宣言→役割2択提示→その場で結論を迫らない。「15万まで上げる方法は？」と聞かれたら「時間ではなく役割で決まる」と答え、経営責任ルートと技術統括ルートの2択で締める。

## 決定事項・アクション
- ヤムズを「完全実装実験機（FC実験1号店）」とする方向で進める。
- 山田に対しては「技術統括／教育専任」ポジションと「実験機責任者」ポジションの2択を提示する。
- その場で結論を迫らず、構造と方向性を提示する面談に留める。
- 加盟金100万B＋ロイヤリティ売上5%＋広告1%をFCテンプレの初期パラメータとして検証する。
- 面談スクリプトを artifact MD として保存し当日参照する。
