# フランチャイズ展開の基本

## ひとことサマリー
直営10店舗→東南アジア16店舗（RIOS INNOVATION）の拡大期に、直営から半FC・FCへの転換論を整理。本部利益最大化を最優先軸に「半FC中心、FCは最小限の選抜型」に着地し、社内会議で意見を引き出すためのPonさん流トーク文まで仕上げた。

## 論点と結論
- FC本部として最初に決めるべき10項目 → ①ブランド核 ②ビジネスモデル数字 ③オーナー像定義 ④オペ標準化（再現性） ⑤教育システム ⑥立地基準 ⑦内装統一 ⑧本部サポート範囲 ⑨契約設計（法務） ⑩拡大スピード
- 失敗しない順序 → 直営完璧→数字固定→マニュアル化→1店FCテスト→修正→本格募集
- 半FC（Semi-FC）の本質 → 「経営参加」。店舗名義・資金は本部、店長は利益分配型責任者、最終責任は本部
- FC（完全FC）の本質 → 「経営移譲」。店舗名義・資金はオーナー、全損益責任、本部はブランド提供者
- 数字比較（月利益40万B例） → 直営本部40万／半FC（30%分配）本部28万＋店長12万／FC（売上10%ロイヤリティ）本部20万
- 本部利益順 → 直営＞半FC＞FC、本気度順 → FC＞半FC＞店長
- Ponさんの選好 → 本部利益最大化が最優先 → 結論「半FC中心、FCは最小限」、FCは拡大装置であって利益最大化装置ではない
- 推奨モデル → 直営7-8／半FC2-3／FC0-1の構成、利益分配25%、材料マージン必須、半FCは「強化型直営」化
- RIOSの未来像 → スター量産ではなく、数追求でもなく、「再現性のある経営者」「攻めながら強くなる会社」「文化を守りながら続く会社」
- フェーズ判定 → Phase1創業者依存（過去）→Phase2幹部育成（半FC・今ここ）→Phase3経営者量産（FC成熟）

## 決定事項・アクション
- 社内会議トーク文（Ponさん風・熱量＋合理性バランス）を生成
- 半FC利益分配率は25-30%レンジを軸に検討
- 材料・カラー剤・店販で別途粗利を取る設計（ロイヤリティ依存しない）
- FC候補は1名のモデルケースのみ実験運用
- 次に詰める候補：具体的な半FC%設計／店長年収納得額／利益1.5倍計算式
