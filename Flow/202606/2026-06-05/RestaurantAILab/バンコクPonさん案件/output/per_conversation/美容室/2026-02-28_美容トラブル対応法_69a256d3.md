# 美容トラブル対応法

## ひとことサマリー
チェンマイ店で欧米客にブリーチ施術中の根元断毛が発生し、4万バーツ請求された案件で、相場感・同意書なしの法的弱さ・誠実→公平→業界基準→提案の交渉設計（15,000B提示→20,000B着地）と、再発防止のための6項目LINE顧客アンケートまで整えた会話。さらにエムクオの両替場所・iPad/iPhone/MacBook 3台 LINE同期方法の質問も含む。

## 論点と結論
- タイでの美容トラブル相場 → 軽度: 無料再施術＋返金30-100%／中度: 全額返金＋ケア代5,000-20,000B／重度: 医療費＋精神補償50,000-200,000B
- 今回ケースの判断 → 医療診断なし＋同意書なし。法的には争えるが Google Review爆撃・欧米コミュニティ拡散リスクがある。妥当ラインは全額返金＋15,000-20,000Bケア補償
- 交渉ステップ（英＋日対訳）→ ①感情の受け止め ②冷静な事実整理（“process”“risk”を使う・あなたのせいと言わない）③公平性（“fairly”/業界基準）④相場ニュアンス挿入 ⑤金額提示 (初手15,000B)。40,000B拒否時は“non-medical”を強調
- 着地戦略 → 初手15,000B→上限25,000B→20,000B着地。レビュー対策は「公開レビューの前に直接ご連絡を」を文中に自然に入れる
- 同意書なし＝法的に弱い → 同時にブリーチ専用英語同意書の即導入を提言
- 顧客アンケート改善 → 「満足度」より「期待値一致」「説明満足度」「施術中不安」を入れて法的防衛＋クレーム予防。シャンプー力加減・トリートメント手触り・価値感などを加え6項目に圧縮した LINE 貼付用版を確定
- エムクオの両替 → Value Plus（Helix Quartier B階）／Emporium地下のOH RiCH Superrich
- 3端末 LINE → iPhone (メイン) ＋ iPad ＋ MacBook (Mac for LINE) で同時利用可能。スマホがログアウトすると全部止まる

## 決定事項・アクション
- 20,000B着地用の柔らかい日本語連結文章（暗にレビュー抑制を含む）を確定
- 6項目アンケート LINE貼付用フォーマットを確定（仕上がり期待値／シャンプー／トリートメント効果／施術前説明／環境／改善要望＋公開レビュー前相談文）
- 今後のために英語同意書／英語フレーズ集／補償ガイドライン／レビュー危機対応マニュアル＝「外国人対応標準プロトコル」整備を経営レベルで進める方針
