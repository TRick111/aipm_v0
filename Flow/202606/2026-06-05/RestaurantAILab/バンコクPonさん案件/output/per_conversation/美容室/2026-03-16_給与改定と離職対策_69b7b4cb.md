# 給与改定と離職対策

## ひとことサマリー
6月の給与改定を前に、Pon さん・岡さん・渡辺さんの3者で行ったラピラビ立て直しミーティングの議事録化依頼。渡辺さんは「給与だけ変えても離職は止まらない／やるなら全部変える」と明言し、モデル店舗運用と歩合制への移行をすり合わせた会話。

## 論点と結論
- 給与改定タイミング → 6月20日または6月末支給分から変えていく方針
- モデル店舗選定 → 大型店のトンロー／サイヤム／アソーク3店舗。1店舗だと歪み、2〜3店舗の方がスタッフトレードもしやすい
- 渡辺さんの核心意見 → 別れたい彼女に化粧・髪型・服を変えるのと一緒。給与体系だけ触っても関係性は戻らない。やるなら雇用契約・遅刻欠勤OTルール・教育・マニュアル・接客基準を全部変える
- 新給与イメージ → 基本給は残しつつ歩合制（アシスタント: シャンプー/ブロー/塗布いくら、カラーリスト: 自分の売上の何%）。「働かないと給料がつかない」設計で“休んだ方が得”の認識を壊す
- 集客の重要性 → 歩合制にしても客が来なければ稼げない。新規獲得・固定客減少改善・リピート率・サービス改善を給与より先に
- サービス方針 → 技術3：サービス7。「ゴミを素通り」「お客様とすれ違っても挨拶しない」など細部をひっくり返す
- 日本人スタッフ問題 → ももさん／トムさん／アイラッシュ2人。9時過ぎてもセット面でメイクしているのに周囲が注意できない→現場ルールが崩壊。モデル不要なら現地スタッフのみ運用が現実的
- 最大ネック → クレジット端末のエラー・二重決済問題。タッチ決済禁止が原則だが守られず、欧米客はQRが使えない
- 離職リスク → ジュノヘア／サランヘアが23,000〜25,000Bレンジで美容師を引き抜き中、特にカラーリストが狙われる
- 一番大きな反省 → 渡辺さんが1年間で多数の問題提起をしたが「問題が起きてから考える」と返され続け、ここ半年は提起をやめていた。小さい不満が積もって現在の離職に繋がった

## 決定事項・アクション
- Pon・岡側で「1店舗か3店舗か」を2日で整理し、渡辺さんに連絡
- 渡辺さん側で受ける場合のイメージ・必要な自由度を引き続き整理
- これからは小さい問題も含めて再度シェアしてもらう（言っても無駄を改める）
- ラプラのKei さん（席数不足で天井問題）への返答を早急に行う（12月から本人が要求中）
- 阿蘇店3月20日離職予定者対応も並行
- 店舗巡回頻度を上げ、現場で即対応する運用へ
