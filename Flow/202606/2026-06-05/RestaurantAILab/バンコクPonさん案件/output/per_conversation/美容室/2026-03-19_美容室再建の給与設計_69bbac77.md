# 美容室再建の給与設計

## ひとことサマリー
ラピラビのトンロー／サイヤム／アソーク3店舗の立て直し責任者として「ワタルさん」に任せる前提で、再建プロジェクト型の給与・KPI・オファー文・ミーティング台本までを最終確定させた会話。

## 論点と結論
- ポジション定義 → エリアマネージャーではなく「再建プロジェクト責任者」。1年契約のプロジェクト型
- 給与構造（最終確定） → 基本給100,000B + 再建プロジェクト手当50,000B = 月150,000B
- 利益配当 → トンロー／サイヤム／アソーク3店舗合計利益の3%を3ヶ月ごと
- 半年KPIボーナス（最大150,000B）→ 売上 (去年6ヶ月平均+20%) 40,000B／離職率 (半年で50%改善) 30,000B／客単価 (+10%) 20,000B／来店周期 (20%短縮 POS管理) 20,000B／コンプリート 40,000B
- 1年完走ボーナス → 100,000B
- 計算ルール明文化 → 売上は去年同期間3店舗平均、離職は自己都合のみ、客単価は総売上÷総来店数、来店周期は前回→今回平均日数（POS算出・リピーターのみ）
- 80%ルール → KPI達成率80%なら80%支給、90%なら90%支給で挑戦しやすい設計
- 90日プラン → Phase1 (0-30日: 見える化＋全員面談)／Phase2 (30-60日: 仕組み化＋単価アップ設計)／Phase3 (60-90日: 店長育成＋安定化)
- 想定年収レンジ → 2,200,000〜2,600,000B

## 決定事項・アクション
- 月給150,000B＋3ヶ月ごとの利益3%＋半年KPIボーナス最大150,000B＋1年完走100,000Bで給与設計確定
- コンプリートボーナスは40,000B（50,000Bから本人指示で調整）
- ワタルさん用オファー文／ミーティング台本／想定Q&A切り返し集を確定し、即ミーティング用に整備
- 「ワタルさん」表記で統一（一度「渡るさん」と誤記したものを訂正）
