# バンコク医療クリニック: 月間売上別の利益シミュレーション

## 前提条件
- 月間売上： **40万バーツ**, **50万バーツ**, **60万バーツ** の3パターン  
- スタッフ構成：施術スタッフ1名（基本給70,000バーツ＋歩合給）、受付スタッフ1名（固定給25,000バーツ）  
- 歩合給（コミッション）率：  
  - 売上 **40万バーツまでは8〜10%**（※本シミュレーションではスタッフのモチベーション向上のため **10%** で計算）  
  - 売上 **40万〜50万バーツ** の範囲：**10%**  
  - 売上 **50万バーツ超過分**：**15%**  
- その他の固定費用：  
  - 店舗家賃：90,000バーツ/月  
  - 警察への支払い：20,000バーツ/月  
  - 医師名義使用料：25,000バーツ/月  
  - WAM広告費：24,000バーツ/月  
  - 施術用ジェル消耗品：9,800バーツ/月  
  - （上記固定費合計：**168,800バーツ/月**）  
- 税金・決済手数料等：売上に対する税金 **7%**（VAT等）およびクレジットカード決済手数料 **2%**  

## シミュレーション結果
上記条件に基づき、**月間売上40万・50万・60万バーツ**の場合の経費内訳と利益シミュレーション結果を以下の表にまとめました。

| 項目                     | 売上40万バーツ | 売上50万バーツ | 売上60万バーツ |
| ------------------------ | -----------: | -----------: | -----------: |
| コミッション（歩合給）     |      40,000 |      50,000 |      65,000 |
| 人件費合計               |     135,000 |     145,000 |     160,000 |
| 固定費合計               |     168,800 |     168,800 |     168,800 |
| 営業利益（税引前）         |      96,200 |     186,200 |     271,200 |
| 税金・手数料              |      36,000 |      45,000 |      54,000 |
| 最終利益（純利益）         |      60,200 |     141,200 |     217,200 |

## 考察・ポイント
- **スタッフの歩合額は売上増とともに大きく増加し、** 40万バーツ時の**40,000バーツ**から60万バーツ時には**65,000バーツ**となります。施術スタッフの月収（基本給＋歩合）も **110,000バーツ** → **135,000バーツ** と上昇し、売上拡大に応じて収入が増えるためモチベーションが高まります。  
- 特に **50万バーツ**を超える部分の売上には高率の**15%**コミッションが適用されるため、**さらに売上を伸ばすインセンティブ**が強化されています。  
- クリニック側の**営業利益・純利益も売上増に伴い増加**します。60万バーツ売上時の純利益は約**217,200バーツ**となり、40万バーツ時の**60,200バーツ**に比べて大幅に増加しています（歩合給を手厚く支給しても、売上拡大により利益総額は拡大します）。
