# Request for reference

## ひとことサマリー
バンコクで美容室併設のネイルサロンをポップアップ形式で立ち上げるにあたり、Instagram運用初期に効くキャプションの2025年トレンド構成・テンプレ5パターン、参考アカウント10件、ハッシュタグ戦略までをスタッフ説明用に体系化した会話。

## 論点と結論
- 2025年現在のInstagramキャプションは「①引き込み一文＋②デザイン/想いの説明＋③会話誘導型CTA」の3層構成が伸びており、絵文字2〜3個と改行でリズムをつけて「ですます調より親しみ調」にすると反応が取りやすいとまとめた。
- ターゲット別5パターン（上品透明感／韓国ちゅるん／Y2Kストリート／ガーリー量産型／ナチュラルタイローカル）のキャプチン雛形を作成し、ポップアップの限定特典や「DM予約」を組み込んだコピペ可能な形にした。
- 参考になるバンコク在住ネイル／ビューティー系アカウント10件（@barenailspa／@hugsxkissesnailbar／@kaianailspa／@abovethenail／@nail_it_th／@artisannails__／@20nailstudio／@glowupbeautyhaus／@yumotohairnailspa／@thesensenailsalon）を抽出し、LINEで送信できる形式に整えた。
- 美容室×ネイルというPonさんのハイブリッド形態には @glowupbeautyhaus と @yumotohairnailspa が最も近く、トータルビューティ／ポップアップ運用の参考モデルとして特に注目すべきだと結論した。
- ハッシュタグは #nailbangkok ／#nailsalonbangkok ／#gelnailsbangkok ／#thainails が地域×サービスで効きやすく、#nails や #nailart のような大競争タグは埋もれやすいため避けるか組み合わせ要素として使うべきだと整理した。
- ハッシュタグ数は10〜15個に絞る、ローカル言語タグ（#ネイルバンコク／#นีลบางกอก）を併用する、サロン独自ブランドタグ（例：#RapirabiPopUpNails）を作るとブランド資産化できると提案した。

## 決定事項・アクション
- スタッフはキャプチン3層構成（引き込み＋説明＋CTA）と5パターン雛形を投稿時の標準フォーマットとして使用する。
- 参考10アカウントをLINEで共有し、各スタッフが投稿構成・写真トーンの研究素材とする。
- 次のステップ案として「英語＋日本語＋タイ語ミックスキャプション5本」「ハッシュタグテンプレ20個」を作成する余地を残した。
