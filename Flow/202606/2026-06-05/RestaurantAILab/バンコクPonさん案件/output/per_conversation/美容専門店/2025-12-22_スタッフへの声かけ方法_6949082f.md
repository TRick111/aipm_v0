# スタッフへの声かけ方法

## ひとことサマリー
タイ移住希望の日本人ネイリスト候補（ギャル気質・ガッツあり・調子に乗りやすい）と Zoom 面談する直前に、希望を煽らず現実を置いて覚悟を確認する6項目台本・ポップアップで判断する仕組み・刺さるフレーズ集・実績ベース立地戦略までを作り込んだ会話。

## 論点と結論
- Zoom 面談のスタンス → 「夢を応援する面談」でも「条件すり合わせ」でもなく、現実を提示して本人に残るか決めさせる場
- 3つの選択肢 → ①ポップアップで結果を出して次に進む ②違うと感じたら終わる ③別会社・別立場で全部自分でやる（一番自由で一番守られない／覚悟がある人専用）
- ポップアップで見る指標 → 事前集客（個人発信で予約作れるか）／当日現場力（時間管理・接客・クオリティ）／単価と満足度／次回予約姿勢
- 危ない反応への即返し → 「集客は店がやってくれますよね？」「とりあえず行けば何とかなる」「条件先に決めたい」「数字は苦手」「サポートあるなら安心」全パターン即返しフレーズ集
- 「次が最後のポップアップ」の伝え方 → 何度もチャンスは作らない、1回で十分わかる、意地悪ではなく現実的判断
- ギャル気質×ガッツ型への寄り添い調整 → 「最初前のめりで話したのは可能性を感じたから」と気まずさを先に処理／勢いを否定せず「行動早い人ほど“設計しないと危ない”」と並走トーン／「見せる側の場」「見られてる感」「選ばれてる感」「覚悟扱い」が効く
- 立地戦略（本人実績から逆算）→ 人形町 yugume 70万単価1万円 (SNS担当あり)／江戸川区ヘアプラス 65万単価7千円 (HPP最上位)／フリーランス時 30万手取りで1年フルブッキング (広告なし)
- 推奨ポジショニング → 単価2,000〜2,500B / 月70〜80人 / バンコク中心 (トンロー・プロンポン・サトーン) の感度高エリア。フリーランス寄り＋設計サポート＝二刀流への助走

## 決定事項・アクション
- Zoom トーク台本は6項目構成 (①前提置く ②動機確認 ③現実提示 ④ポップアップ説明 ⑤分岐提示 ⑥腹括れるか確認) で確定
- 最後の問いは「やる？」ではなく「ここまで聞いて正直どう感じた？」
- ギャル気質タイプ専用「刺さるフレーズ集」を作成
- 立地は単価で選ばれる人なので「安さで選ばれる場所」ではなく「人で選ばれる場所」に絞る方針
