# archive/ — 廃ページ・歴史的アーカイブ

## ここに置くもの
不要になったが歴史的に残すページ

## ここに置かないもの
active なものは該当 dir に

## ファイル命名
`<slug>.md`（小文字、ASCII or 日本語、空白はハイフン）

## 必須 frontmatter
`slug` / `type` / `created_at` / `updated_at`（その他 type 固有フィールドは schema.md 参照）
