# meetings/ — 特定日時のイベント記録

## ここに置くもの
日付指定の会議・イベント。出席者と決定事項

## ここに置かないもの
日付なしの discussion は writing/ または concepts/

## ファイル命名
`<slug>.md`（小文字、ASCII or 日本語、空白はハイフン）

## 必須 frontmatter
`slug` / `type` / `created_at` / `updated_at`（その他 type 固有フィールドは schema.md 参照）
