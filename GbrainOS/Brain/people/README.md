# people/ — 人物（個人）

## ここに置くもの
1 ファイル = 1 人。同一人物は alias でまとめる

## ここに置かないもの
組織として記述するなら companies/

## ファイル命名
`<slug>.md`（小文字、ASCII or 日本語、空白はハイフン）

## 必須 frontmatter
`slug` / `type` / `created_at` / `updated_at`（その他 type 固有フィールドは schema.md 参照）
