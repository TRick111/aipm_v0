# ビリヤード俯瞰アプリ

斜めから撮ったビリヤード台の写真を、上から見た図（俯瞰図）に変換するWebアプリ。

## 背景・課題

ポケットビリヤードのプレイ中に、斜めから台の写真を撮影し、それを俯瞰図に変換したい。
以前の自作の試みで以下まではできた：

- ラシャの色検出 → 台のエッジ抽出 → 平面の取得

**未解決の課題：**
- ボールの検出
- 斜めから見たボールの画像内座標を3D的に再構成して平面に戻す処理

## ゴール

写真1枚から、全ボールの正確な俯瞰位置を復元して描画するWebアプリ

## 現在の状況・ネクストアクション
- 現状: 超荒いプロトタイプが完成。ただしボール位置の検出精度が非常に低い
- 優先度: **低**（個人の趣味プロジェクト）
- ネクストアクション:
  - [ ] 内部の仕様（現在の検出ロジック）を理解・整理する
  - [ ] ボール位置の精度向上のための手法を調査する
  - [ ] 調査結果をもとに精度改善を実装する
