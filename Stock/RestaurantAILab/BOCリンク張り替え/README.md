# BOCリンク張り替え

## 背景
- クライアント「ベーカリーBOC（Bridge on the Cloud）」からサイト内のページURLの書き換えを依頼されている

## 目的
- 指定されたページのURLリンクを正しいものに書き換える

## ゴール（完了条件）
- 書き換え対象のリンクがすべて正しいURLに更新されていること
- クライアントに完了報告が済んでいること
