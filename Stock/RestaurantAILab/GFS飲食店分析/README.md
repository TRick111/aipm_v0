# GFS飲食店分析

## 背景
- **GFS**（依頼元）は飲食コンサルタントを複数抱え、飲食店にコンサルティングサービスを提供している会社
- GFSがクライアント飲食店へのコンサル品質向上を目指しており、売上データと口コミの分析をAIで支援し、データドリブンな提案を実現する

## 目的
- GFSの顧客である飲食店の売上データと口コミを分析し、経営改善に向けた示唆を提供する

## ゴール（完了条件）
- GFSクライアント飲食店の売上・口コミデータから実用的な改善提案を導出し、コンサルに活用できる状態にする

## 成果物一覧

### THE BIFTEKI 赤坂見附店
- **売上分析中間レポート** (`THE BIFTEKI 赤坂見附店/reports/THE_BIFTEKI_赤坂見附店_売上分析中間レポート.md`)
  - 売上全体推移、客数分析、客単価分析、回転率分析、セグメント分析の統合レポート
  - **主要結論**: 2025年の売上増加は「客数増」と「客単価上昇」の両方によるが、**客単価の伸び率が客数の約3倍（+40% vs +14%）**
  - 売上向上の最重要ポイント: **ディナー帯の集客強化**（売上との相関r=0.926）

- **スライド構成ドラフト** (`THE BIFTEKI 赤坂見附店/slidedraft/2026-01-28/スライド構成_ドラフト.md`)

  - [客数分析レポート](THE%20BIFTEKI%20赤坂見附店/reports/customer_count_analysis_report.md)

- **主要インサイト**
  - 平日はディナー来店数が売上を決定（相関0.926）、土日は総来店客数（相関0.961）
  - 客単価: 2024年→2025年で**+40%上昇**（1,189円→1,661円）
  - 高単価メニュー戦略が成功（サンシリーズ売上構成比18.3%）
