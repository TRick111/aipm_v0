# GFS飲食店分析

## 背景
- 飲食コンサルティング会社GFSがクライアント飲食店へのコンサル品質向上を目指している
- 売上データと口コミの分析をAIで支援し、データドリブンな提案を実現する

## 目的
- GFSの顧客である飲食店の売上データと口コミを分析し、経営改善に向けた示唆を提供する

## ゴール（完了条件）
- GFSクライアント飲食店の売上・口コミデータから実用的な改善提案を導出し、コンサルに活用できる状態にする
