# Restaurant AI Lab

## 背景
- 飲食業界ではDX化の遅れが課題となっており、業務効率化やマーケティング強化のニーズが高まっている
- 飲食コンサル（Chef's Room）と連携し、現場知見とAI技術を組み合わせたサービス提供が可能になった

## 目的
- 飲食店向けにAIを活用したサービス（口コミ分析、マーケティング支援、業務効率化等）を提供する
- 飲食コンサルと協業し、実用的で導入しやすいソリューションを展開する
