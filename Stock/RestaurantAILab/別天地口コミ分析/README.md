## 主な分析結果（サマリー）

### 強み
- **松坂牛しゃぶしゃぶ**：一人一鍋スタイルが圧倒的強み
- **金目鯛の煮付け**：サイズのインパクトと上品さが好評
- **カラスミ蕎麦**：目の前で削るパフォーマンスが「感動」を生む

### 課題
- **和牛ヒレカツサンド**：口コミ評価は極めて高いが売上4個のみ → 認知度向上が急務
- **アラカルト単品**：コース利用メインのため認知度が低い
