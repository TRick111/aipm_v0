# 月報

## 背景
- クライアント飲食店に対して月次の売上・運営レポートを提供するニーズがある
- 週報とは別に、月単位の俯瞰的な分析・報告を行う

## 目的
- クライアント飲食店の月次データを分析し、月報として提供する
- 週報のデータを月単位で再集計し、トレンド分析・前年比較・週別深掘りを行う

## ゴール（完了条件）
- 各クライアント（別天地、BFA等）の月報が定期的に作成・納品されている
- 月報作成プロセスがプロンプト+スクリプトで再現可能な状態
