## 【スライド構成の補足説明】

### 設計の考え方

1. **1スライド1主張**: 各スライドで伝えたいことを1つに絞り、メッセージを明確にしています

2. **ビジュアル重視**: 数値分析に慣れていない経営者向けに、表、グラフ、比較ボックスなど視覚的に分かりやすい表現を採用

3. **ポジティブ・ネガティブのバランス**: 成長を強調しつつ、課題も明確に提示

4. **具体的なアクション**: 「何をすべきか」が明確で、すぐに実行できる内容

5. **段階的な説明**:
   - サマリーで全体像
   - 詳細分析で深掘り
   - 総括で振り返り
   - アクションで次につなげる

### プレゼンテーションの流れ

```
導入（5分）
 ├─ タイトル・目次
 └─ 今週のサマリー

本論（20-25分）
 ├─ 売上分析（10-12分）
 ├─ 商品分析（3-4分）
 └─ 総括と課題（3-4分）

結論（5-8分）
 ├─ アクションプラン
 └─ 目標と次週

質疑（5-10分）
```

**合計時間**: 35-45分程度
