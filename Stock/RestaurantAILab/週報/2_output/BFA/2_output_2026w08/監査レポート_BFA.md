### G3: page-break設定 → 要確認

**検証内容**:

```css
/* ベース設定 */
.slide {
  page-break-after: always;  /* レガシー ✓ */
  break-after: page;         /* モダン ✓ */
}

/* 最終スライド */
.slide:last-child {
  page-break-after: auto;    /* レガシー ✓ */
  /* break-after: auto; ← 未設定 */
}
```

ベースの.slideルールでは両プロパティが設定されているが、`:last-child`ルールではpage-break-after:autoのみで、break-after:autoが未設定。実用上はpage-break-afterが広くサポートされており表示上の問題はないが、プロパティの整合性の観点で`break-after:auto`の追加を推奨。
