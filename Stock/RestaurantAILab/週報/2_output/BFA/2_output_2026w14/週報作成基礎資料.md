## 4. 総合的な考察とアクションプラン

### 4.1 現状総括

**強み**:
- 客単価が¥5,889と高水準を維持（前週比+21.5%、前年比+11.7%）
- 法人貸切・送別会需要の取り込み実績（24名の部門送別会、過去にも同企業の貸切利用あり）
- Google検索経由の海外ゲスト・新規来店が定着しつつある
- 森のジントニックやタリスカー10年など、商品力のあるメニューが育っている
- スタッフ（大泉さん）が来店経路・嗜好のヒアリングを丁寧に実施

**課題**:
- 客数が前年比-47.7%と大幅減。売上も前年比-41.6%
- 平日（特に木曜日）の集客力不足。木曜は6人しか来店がなかった
- 土曜日も4週平均比-¥57,320と不振。週末の安定した集客ができていない
- 口コミ投稿がゼロで、オンラインでの認知拡大が停滞
- 18-20時の早い時間帯がほぼ空白で、営業時間の活用が非効率

**優先順位**（深刻度×影響度）:
1. 客数回復（全曜日共通）— 売上回復の最重要課題
2. 平日集客の強化（特に木曜日）— 安定した売上基盤の構築
3. 週末（土曜）の安定化 — 本来稼ぐべき曜日の立て直し
4. 口コミ・レビュー促進 — 中長期の集客基盤

### 4.2 アクションプラン

**短期施策（1-3ヶ月）**:
1. Google検索対策の強化: Googleビジネスプロフィールの更新頻度を上げる（写真追加、投稿）
2. 平日限定施策: 木曜日のハッピーアワーやスタッフおすすめカクテルのSNS告知
3. 口コミ促進: 来店客への口コミ投稿の声かけ、QRコード付き案内カードの設置
4. 海外ゲスト対応強化: 英語メニューの整備（日報で課題として指摘あり）
5. チョコレートのオペレーション改善: ピーク前補充の徹底（日報で課題認識済み）

**中期施策（3-6ヶ月）**:
1. 法人宴会営業の体系化: 過去の貸切実績企業へのDM、法人向け宴会プランの整備
2. 予約管理の強化: ホットペッパー等の予約サイトの活用拡大
3. 甘くないカクテルの充実: 日報で「甘くないカクテルの提案に困った」との課題あり。レシピリスト作成
4. 早い時間帯（18-20時）の活性化: フード中心のディナーメニューや早割施策の導入

**長期施策（6ヶ月-1年）**:
1. ブランドポジショニングの明確化: 「Google検索で見つかるバー」としてのオンラインプレゼンス強化
2. リピーター育成プログラム: 常連客のデータベース化と来店頻度の向上施策
3. メニューの構造改革: 売れ筋（森のジントニック、ウイスキー）を軸とした看板メニューの確立
