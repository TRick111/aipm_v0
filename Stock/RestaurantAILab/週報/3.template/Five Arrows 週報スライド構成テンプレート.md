# Five Arrows 週報スライド構成テンプレート（詳細版）

本テンプレートは、週次営業報告スライド（30枚構成）の標準フォーマットです。  
毎週のデータを入れ替えることで、一貫性のある週報を作成できます。

## 使用上の注意

1. **深掘り対象曜日の選定**：各週で最も好調な曜日と最も不調な曜日を選定し、深掘り分析を行う
2. **データの一貫性**：すべての比較は「過去4週平均」を基準とし、前年同週比較は第1部のトレンド分析で行う
3. **1スライド1メッセージ**：各スライドは1つの明確なメッセージを伝えることに集中する
4. **ビジュアルの統一**：色使い、フォント、レイアウトは全スライドで統一する
5. **施策一覧の更新**：施策一覧は週ごとに更新し、進捗状況を反映する
6. 重要！！　グラフの描画に必要な数値を、すべて記載内容（詳細）の中に含めること
