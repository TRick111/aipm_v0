# Manus Nanobananaスライド デザイン方針ガイド

## 1. はじめに

本ドキュメントは、コンサルティングレポート（PDF）のデザインを、ManusのNanobananaスライド（画像生成モード）を用いて再現するためのデザイン方針を定義するものです。主な読者である飲食店の店長様が直感的かつ迅速に内容を理解できるよう、シンプルで視覚的なコミュニケーションを重視したスライドデザインを目指します。

### 基本原則

1. **情報の簡潔性**: 1スライドには1つの主要なメッセージのみを盛り込み、情報を詰め込みすぎない
2. **段階的な説明**: 複雑な内容は複数のスライドに分割し、段階的に理解を促す
3. **焦点の明確化**: グラフやデータを含むスライドでは、最も注目すべき点を視覚的にハイライトする

---

## 2. Nanobananaスライドの特性と活用

Nanobananaスライドは、各スライドを一枚の高解像度画像として生成するManusの機能です。

| 特性 | 詳細と活用方針 |
| :--- | :--- |
| **スタジオ品質のビジュアル** | プロのデザイナーが作成したような高品質な画像やイラストを生成可能。ストックフォトにはない、オリジナリティと一貫性のある世界観を構築できる |
| **テキストの高精度レンダリング** | AI画像生成で課題となりがちなテキストの描画精度が非常に高い。デザイン性の高いタイポグラフィを画像内に直接、鮮明に埋め込み可能 |
| **統一されたアートディレクション** | プロンプトに基づいて、スライドデッキ全体で一貫したビジュアルスタイル（色、フォント、雰囲気）を維持 |
| **生成後の編集可能性** | 画像生成後でも、テキストの修正や特定箇所の再描画といった「ポイント＆エディット」が可能 |

## 3. デザイン要素の再現方針

### 3.1 カラーパレット

元のデザインで使用されているカラーパレットは、情報の種類を直感的に伝える役割を果たしています。

| 用途 | 色 | HEXコード | プロンプト指示例 |
| :--- | :--- | :--- | :--- |
| **主要カラー（ヘッダー等）** | ダークネイビー | `#1E3A5F` | 「ダークネイビーの背景」「信頼感のあるプロフェッショナルな雰囲気」 |
| **ポジティブ・達成** | グリーン | `#22C55E` | 「ポジティブな数値は明るいグリーンで」「達成を#22C55Eでハイライト」 |
| **注目・変化** | オレンジ | `#F97316` | 「強調部分は鮮やかなオレンジで」「変化を#F97316でハイライト」 |
| **背景** | ライトグレー | `#F1F5F9` | 「清潔感のあるライトグレー背景（#F1F5F9）」 |

### 3.2 タイポグラフィとテキスト

飲食店店長様向けに、可読性とインパクトを両立させます。

- **数値の強調**: 「数字『4.2』を非常に大きく、太字で、グリーンのフォントで表示」のように、サイズ・太さ・色を具体的に指定
- **シンプルな言葉選び**: 専門用語を避け、「売上」や「お客様の声」といった平易な言葉でプロンプトを作成
- **情報量の制限**: 1スライドあたりのテキストは、タイトル、キーメッセージ、2〜3の補足情報に留める

### 3.3 レイアウトと構成

元のデザインに見られる2カラムレイアウトやカード形式は、情報を整理し、視線を誘導するのに効果的です。

- **2カラム**: 「2カラムレイアウトで作成。左カラムは定量データ、右カラムは定性分析」
- **カード形式**: 「スライド下部に3つの主要指標を別々のカードで表示」
- **焦点の明確化**: 「問題から解決策への結果を示す大きな矢印を使用」のように、視覚的な誘導を指示

### 3.4 グラフとデータの可視化

グラフの「どこを見るべきか」を明確にすることが最も重要です。

- **シンプルなグラフ**: 「過去6ヶ月のトレンドを示すシンプルな折れ線グラフを生成」
- **ハイライト**: 「棒グラフで『12月15日』の棒をハイライトし、『SNSキャンペーンで来客数急増』と吹き出しを追加」
- **色分け**: 「複合グラフで売上はグリーン、来客数はブルーを使用」

## 4. スライド作成プロンプト例

### 例1：口コミ状況分析レポート

```
飲食店店長向けの「2025年12月 口コミ分析レポート」スライドを作成。

- ヘッダーはダークネイビー背景（#1E3A5F）
- メインコンテンツエリアはライトグレー背景（#F1F5F9）
- 2カラムレイアウトを使用

【左カラム：定量データ】
- 大きなグリーンの星アイコンを表示
- その横に「4.2」を非常に大きく、太字で、明るいグリーンのフォントで表示
- スコアの下に「（前月比 +0.3）」を小さめのオレンジフォントで表示
- その下に過去6ヶ月（7月〜12月）の上昇トレンドを示すシンプルな折れ線グラフ

【右カラム：定性分析】
- 「良い評価」セクションに「スタッフが親切」「新メニューが美味しい」「清潔感がある」をグリーンの丸みを帯びたタグで表示
- 「改善された課題」セクションを作成。以前の課題「提供スピードが遅い」が解決されたことを示す
- 結果として「今月のクレーム数：0」を表示。大きなオレンジの下向き矢印で指し示し、「0」を非常に大きくオレンジで表示
```

### 例2：経営指標ダッシュボード

```
飲食店店長向けの「2025年12月 経営ダッシュボード」スライドを作成。

- ヘッダーはダークネイビー（#1E3A5F）
- メインコンテンツエリアはライトグレー背景（#F1F5F9）

【メインダッシュボード：売上と来客数トレンド】
- 12月の複合グラフを生成。「売上」は折れ線グラフ、「来客数」は棒グラフで表示
- データは12月15日頃に顕著なスパイクを示す
- 12月15日のスパイクを指すダークブルーの吹き出しボックスを追加。テキストは「SNSキャンペーンにより来客数急増」

【下部のサブKPI】
- メインチャートの下に3つの主要指標を、清潔感のある別々のカードで表示
- カード1：「客単価」の値「4,200円」を大きなグリーンフォントで
- カード2：「リピート率」の値「35%」を大きなグリーンフォントで
- カード3：「ウェブ予約率」の値「60%」を大きなグリーンフォントで
```
