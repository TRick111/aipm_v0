# 振り返り

## 背景
- 日々の出来事や感情を振り返る習慣を持つことで、自己理解を深め、精神的な安定を図りたい
- 短期（日次）・中期（週次）・長期（月次）の視点で自分を見つめ直し、成長を実感できるようにする

## 目的
- 日々の振り返りを通じて、感情・行動・学びを言語化する
- 週次・月次の振り返りで、パターンや傾向を把握し、改善につなげる
- 成長の軌跡を記録し、自己肯定感を高める

## ゴール（完了条件）
- 日次振り返りの習慣が定着している
- 週次・月次の振り返りテンプレートが確立し、運用されている
- 振り返りを通じて気づきや改善アクションが生まれている
