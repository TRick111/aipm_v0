# 健康管理

## 背景
- 体重が増加傾向にあり、体脂肪率も上がってきている
- 日々の運動・食事を意識的に管理し、改善したい

## 目的
- 体重・体脂肪率を健康的な水準に戻す
- 運動習慣と食事意識を定着させる
- 睡眠の質を記録・可視化し、睡眠の質を向上させる

## ゴール（完了条件）
- （体重・体脂肪率の目標値が決まり次第記入。未定でも運用開始してOK）
- 睡眠データがデバイスからAIOSへ自動連携され、日々の睡眠スコアが記録されている状態（2026-04末目標）

## 指標
- 体重 (kg)
- 体脂肪率 (%)
- 睡眠スコア / 睡眠時間 / 深い睡眠の割合（デバイス導入後）

## 日次で記録すること
- 体重（測れた日のみ）
- 運動の有無・内容
- 食事メモ（気をつけたこと等。任意）
- 睡眠データ（デバイス導入後、自動連携予定）
