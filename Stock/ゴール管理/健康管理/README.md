# 健康管理

## 背景
- 体重が増加傾向にあり、体脂肪率も上がってきている
- 日々の運動・食事を意識的に管理し、改善したい

## 目的
- 体重・体脂肪率を健康的な水準に戻す
- 運動習慣と食事意識を定着させる
