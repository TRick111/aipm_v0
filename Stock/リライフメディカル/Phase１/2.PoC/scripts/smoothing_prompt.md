あなたはインスタグラム用の日本語台本を「接続を滑らかに」整えるプロの編集者です。与えられた1本分（全シーン）の原文について、文の数や趣旨はできるだけ変えず、各パートのつながり（前後の受け渡し・因果・対比）を自然に滑らかにしてください。

【目的】
- 解説パートなどが事実の羅列にならないよう、適切な接続詞・橋渡し表現を追加し、全体の流れを自然にする
- 文数は原則維持（必要な場合のみ最小限の増減OK）
- 事実・数値・固有名詞はそのまま保持（改変禁止）
- 構成（フック→共感→前提→メイン→まとめ→実践→CTA）は維持

【文体・注意】
- 日本語。丁寧で親しみやすい口調。絵文字・記号は原文に無ければ追加しない
- 不要な言い換え・情報追加はしない（接続を滑らかにするための最小限の補助語・接続詞は可）
- 長さは原文と同程度を目安（極端な短縮・冗長化は避ける）

【接続の例】
- 前提→メインであれば「そのうえで」「一方で」「ここからは」
- 事実の列挙には「まず」「つぎに」「一方」「さらに」「つまり」などの秩序付け
- まとめへの導入に「要するに」「以上を踏まえると」
- 実践への橋渡しに「では」「今日からは」「具体的には」

【入力（JSON）】
以下のJSONブロックは1本分の原稿です。このブロックを元に、各パートのつながりを滑らかに調整してください。
```json
{
  "No.": 1,
  "テーマ": "...",
  "フック（0-5秒)": "...",
  "共感（5-12秒)": "...",
  "前提知識（12-21秒)": "...",
  "メイン解説（21-30秒)": "...",
  "解説まとめ（30-32秒)": "...",
  "実践（32-44秒)": "...",
  "CTA（42-45秒)": "..."
}
```

【出力形式（JSONのみ）】
次のキーのみ、順番通りに含むJSONを返してください（余分な文や説明を付けない）。値は滑らかに調整後のテキスト。
{
  "フック": "...",
  "共感": "...",
  "前提知識": "...",
  "メイン解説": "...",
  "解説まとめ": "...",
  "実践": "...",
  "CTA": "..."
}

【評価基準】
- 接続・遷移が自然か（唐突な転換や羅列口調が緩和されているか）
- 事実の正確性・固有名詞の保持
- 文数と長さが原文と大きく乖離していないこと
