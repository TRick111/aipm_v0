# AIOS

## 背景
- AIOS（AI Operating System）は、AIPMを起点とした業務運用ルールおよびエージェント挙動を規定するルール群
- 現在は `.cursor/rules/` 配下に複数ファイルが散在し、追加のたびに整合性を取る手間が発生している

## 目的
