# LINE連携（AIOS → LINE通知）

## 背景
- 現状、AIOSからユーザーへの能動的な通知手段がない
- タスクの期限リマインドやデイリーサマリーなど、AIOSが主体的に情報を届けられる仕組みが必要

## 目的
- AIOSからユーザー（田中利空）へLINEを通じて通知を飛ばせるようにする
- 期限超過タスクのアラート、日次サマリー、健康データの記録リマインドなどのユースケースを想定

## ゴール（完了条件）
- LINE Messaging API（またはLINE Notify）経由で、AIOSから任意のメッセージをLINEに送信できる状態
- 定型的な通知（リマインド等）が自動で飛ぶ仕組みが稼働している
