# マンション内LAN

## 背景
- 現住居（集合住宅）に引っ越して以降、自宅 Wi-Fi が「甘い」状態が継続
- パソコン・スマートフォン双方で症状が発生（携帯はセルラー通信なら高速）
- 「あるページにその日最初にアクセスする時は遅く、2回目以降は軽くなる」という再現性のある症状
- 2026-04-22 の診断（`環境整備/ネットワーク最適化診断_2026-04-22.md`）で MTU 1024 へ下げると改善することは判明済
- ただし MTU 1024 は他ソフトに影響があるため恒常的には適用したくない
- 2026-05-04 に再診断を実施し、ICMP フィルタ・ダブル NAT・TCP SYN 再送など新しい所見を取得

## 目的
- 自宅 Wi-Fi の体感速度劣化の真因を特定する
- MTU を 1500 のまま、他ソフトに影響を与えずに改善する打ち手を確立する
- マンション側設備に起因する制約と、自分側で打てる手の境界を明らかにする

## ゴール（完了条件）
- 「初回アクセス遅延」が体感で消える、または許容範囲まで改善している
- 適用した対策と効果が log として残っており、引っ越し時・設備変更時に再現可能
- マンション管理会社への問い合わせが必要な事項が整理され、判断ができる状態
