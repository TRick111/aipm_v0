# 会議議事録の自動反映

## 背景
- オンライン・オフライン問わず、会議終了後にその文字起こしをAIPMに反映する作業が発生する
- 手動での反映は漏れが生じやすく、効率化が必要

## 目的
- 会議終了後の文字起こしを、AIPMのワークスペース（Meetings / 関連プロジェクトのREADME等）に自動的に反映する仕組みを構築する

## ゴール（完了条件）
- 会議の文字起こしデータをインプットとして、AIPMへの反映が自動 or 半自動で行えること
