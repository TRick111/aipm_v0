# 定型作業

## 背景
- 会社カレンダーを社外に共有できない制約がある一方で、社外の関係者と日程調整を進めるには「自分の空き/空いてない時間」をGoogleカレンダー等に反映して運用する必要がある。
- 提携で行う作業は、手順・参照ファイル・運用ルールが散逸しやすく、再現性/引き継ぎ/自動化の観点で集約が必要。

## 目的
- 提携で行う「定型作業」のやり方（手順・参照ファイル・注意点）をStockに蓄積し、必要に応じてエージェントが参照できる状態にする。
- MCP等を活用し、定型作業の一部を自動化できる基盤（依頼→実行→記録）を整える。
