# Backlog

目的: 自分が実施するタスクを横断管理し、**優先度 / 期日 / 所属（Program/Project）** を一覧で把握する。

運用ルール（機械処理しやすくするため）:
- **列は固定**（ヘッダ名・順序を変えない）
- `Due` は `YYYY-MM-DD`（未定は空欄でOK）
- `Priority` は `P0/P1/P2/P3`（P0が最優先）
- `Status` は `todo/doing/blocked/done`
- セル内で `|` を使わない（Markdown表が崩れるため）
- `Tags` は `tag1,tag2` のカンマ区切り（空欄OK）
