# TCS退職手続き

## 背景
- 田中利空はタタコンサルタンシーサービシズ ジャパン(TCS Japan) を退職。現在有給消化中
- 副業として既に開業届を提出済み（個人事業主）→ 有給消化終了後はフリーランス専業に移行
- 「退職した会社員」から「フリーランス専業」への切り替えに伴い、保険・年金・税金・雇用保険などの一般的な手続きを漏れなく実施する必要がある

## 目的
- 退職に伴う公的手続き・社内手続き・私的手続きを期限内に完了させる
- 漏れ・遅延による不利益（保険無加入期間、税金滞納、企業型DC自動移換手数料 等）を防ぐ

## ゴール（完了条件）
- 退職日以降に必要な公的手続き（健康保険・年金・住民税・雇用保険）がすべて完了している
- 会社からの受領書類（離職票・源泉徴収票・退職証明書 等）をすべて受領済み
- 会社への返却物・データ削除をすべて完了
- 企業型DCの移換が完了している
- 翌年の確定申告に必要な書類が揃っている
