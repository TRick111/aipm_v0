## 1. 会社からの受領書類（必須・漏れ厳禁）

会社には発行義務があるもの／離職後の各種手続きに必須。

- [ ] **離職票-1・離職票-2**（雇用保険被保険者離職票）— 失業給付/再就職手当の手続きに必要。退職後10日前後で郵送が一般的
- [ ] **源泉徴収票**（給与・退職金の両方）— 確定申告に必要。退職後1ヶ月以内に発行義務
- [ ] **雇用保険被保険者証**（在職中に会社預かりの場合）
- [ ] **年金手帳 or 基礎年金番号通知書**（会社預かりの場合）
- [ ] **健康保険資格喪失証明書**（任意継続を選ばない場合でも国保加入に必要）
- [ ] **退職証明書**（任意。賃貸・ローン契約等で求められることがある）
- [ ] **企業型DC（確定拠出年金）の脱退一時金 or 移換に関する案内書類**
- [ ] **退職所得の受給に関する申告書**の控え（提出済みの場合。退職金の源泉徴収に影響）

## 3. 健康保険の切り替え（退職日翌日から原則14日 or 20日以内）

3択。**保険料を比較して必ず期限内に決断**すること。

- [ ] **選択肢A：任意継続被保険者制度**（旧健保を最大2年継続）
  - [ ] 退職日翌日から **20日以内** に協会けんぽ/健保組合へ申請
  - [ ] 保険料は全額自己負担（労使折半が無くなる）→ 上限あり
- [ ] **選択肢B：国民健康保険（国保）**
  - [ ] 退職日翌日から **14日以内** に市区町村役場で加入手続き
  - [ ] 健康保険資格喪失証明書 + マイナンバー + 本人確認書類を持参
- [ ] **選択肢C：家族の被扶養者になる**（家族の年収要件・自身の見込み年収要件あり。フリーランスで稼ぐ場合は非該当が普通）
- [ ] 任意継続 vs 国保 の **保険料試算** を実施（年間で数十万円の差が出ることが多い）
- [ ] 決定後、旧健保証を会社に返却（または郵送）

## 4. 年金の切り替え（退職日翌日から14日以内）

- [ ] **厚生年金 → 国民年金（第1号被保険者）への切替** — 市区町村役場の年金窓口で手続き
- [ ] 必要書類：年金手帳/基礎年金番号通知書、退職日が分かる書類（離職票や資格喪失証明書）、本人確認書類、マイナンバー
- [ ] 配偶者が第3号被保険者だった場合は第1号への変更も同時申請
- [ ] **付加年金（月+400円）への加入を検討** — 個人事業主のみ可・コスパ良
- [ ] **国民年金基金 or iDeCoの追加検討**（節税効果あり。次項参照）

## 5. 企業型DC（確定拠出年金）の移換 ★放置厳禁

**退職日翌月から6ヶ月以内に手続きしないと国民年金基金連合会へ自動移換され、手数料発生＋運用停止**。

- [ ] 会社経由で **企業型DCの加入有無** を確認
- [ ] 加入していた場合：
  - [ ] **iDeCo（個人型確定拠出年金）への移換** が原則（フリーランスは月最大68,000円拠出可・全額所得控除）
  - [ ] 金融機関（SBI証券・楽天証券・マネックス等）でiDeCo口座開設
  - [ ] 「個人型年金加入申出書 兼 移換届」を提出
  - [ ] 移換完了まで2-3ヶ月かかる

## 6. 雇用保険・失業保険関連 ✅ 完了 (2026-06-23)

**重要**：個人事業主として開業届を出している場合、原則「失業の状態」にあたらないため失業給付は受給不可。ただし廃業→再就職活動とする選択肢、再就職手当の検討余地あり。
詳細は既存調査レポート `../雑事/unemployment_insurance_investigation.md` を参照（BL-0015 で確定済み）。

- [x] 離職票-1・離職票-2 を受領
- [x] 個人事業主継続のまま受給するか / 廃業届を出して受給するか を判断
  - [x] 税理士 or 社労士に **廃業 → 再開業時の税務影響** を相談
  - [x] 管轄ハローワークに事前相談（個人事業主継続が「就職」扱いか確認）
- [x] 判断後、ハローワークで求職申込み → 受給資格決定（受給する場合）
- [x] 受給しない場合でも離職票は **手元保管**（後日必要になる可能性あり）

## 7. 税金関連

### 7.1 住民税

- [ ] 退職時期による徴収方法を確認
  - **1〜5月退職** → 残額を最終給与から **一括徴収**（会社が自動処理）
  - **6〜12月退職** → 残額を **自分で納付（普通徴収）** に切替（会社経由で自治体に届出）
- [ ] 退職翌年6月以降の住民税は、確定申告に基づき自治体から納付書が届く（普通徴収）

### 7.2 所得税（退職年の確定申告）

- [ ] 退職年の **給与所得 + 事業所得** を合算して確定申告（年末調整なし）
- [ ] 退職金は「退職所得の受給に関する申告書」を出していれば源泉分離課税で完結
- [ ] 翌年 **2/16〜3/15** に申告・納付
- [ ] **青色申告継続**（既に承認申請済みのはず・要確認）
- [ ] 医療費・寄附金・小規模企業共済掛金 等の控除書類を集約

### 7.3 マイナンバー関連

- [ ] e-Tax の利用者識別番号を確認 / マイナポータル連携を準備

## 8. フリーランス専業移行に伴う追加タスク

既に開業届は提出済み。専業化に伴う棚卸し。

- [ ] **開業届の記載内容を確認**（事業所所在地・屋号・事業内容に変更があれば訂正届）
- [ ] **青色申告承認申請書** の提出済み確認（未提出なら開業から2ヶ月以内に提出）
- [ ] **小規模企業共済への加入を検討**（掛金全額所得控除・退職金代わり・月1,000-70,000円）
- [ ] **国民年金基金 or iDeCo の上乗せ拠出を検討**（節税）
- [ ] **インボイス制度（適格請求書発行事業者）登録の判断**（取引先要件次第）
- [ ] **事業用銀行口座・クレジットカードの整備**（既存なら継続利用でOK）
- [ ] **会計ソフトの継続契約**（freee/マネーフォワード等）
- [ ] **国民健康保険組合（業種別）への加入検討**（IT/文芸美術等。所得連動の国保より安くなる場合あり）
